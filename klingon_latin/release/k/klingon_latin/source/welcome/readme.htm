Klingon Latin 1.0 is a Latin-script keyboard for Klingon that supports the use of case, digraphs, and trigraphs. The keyboard layout is based upon the "xifan hol" layout. The mobile keyboard layout has longpress support for Tak'ev and Morska dialect letters, as well as English. The desktop keyboard uses "xifan hol" layout, Shift to select Tak'ev and Morskan letter variants, and Alt and Alt+Shift to access the standard English alphabet.
