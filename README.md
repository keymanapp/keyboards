# Tawrã-Kaman Keyboard
This keyboard is designed for the Tawra (Digaro Mishmi, Taraon)and Kaman (Miju Mishmi) languages of Arunachal Pradesh, India. It provides a comprehensive solution for typing the unique orthography of these languages on both Desktop and Mobile platforms.

## Keyboard Features
* **Smart-Transform Logic:** Automatically combines base vowels with tones and nasalization marks to prevent visual glitches.
* **Special Characters:** Direct access to `ü` and `ə`.
* **Tone Support:** Supports High, Low, and Mid tones via intuitive modifier keys.
* **Mobile Optimized:** Features Auto-Shift and double-tap Caps Lock for faster mobile typing.

## Desktop Layout
* **ü / ə**: Use `[` and `]` keys.
* **Tones**: Type the vowel first, then use `` ` `` (Backquote) for High tone, `~` for Low tone, and double-press `` ` `` for Mid tone.
* **Circumflex**: Use `Shift` + `6` after a vowel.
* **Nasalization**: Use `\` after a vowel.
