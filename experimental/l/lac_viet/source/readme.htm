Lac Viet

Lac Viet keyboard for Vạ Xoạng (Zhuang Language in Chữ Quốc Ngữ) Chữ Quốc Ngữ tiếng Choang.國語字壯文

This script is a innovative romanization form that adopted Chữ Quốc Ngữ system to represent the Zhuang or Choang speak. This script will be shorter, more readable and more practical than previous "vahcuengh" romanization system.
