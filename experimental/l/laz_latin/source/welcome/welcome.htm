This is a Laz Latin keyboard for transcription of the Lazuri language.

Keyboard Layout

Desktop Keyboard layout

Press the circumflex key (SHIFT 3) followed by a vowel to get a circumflex above a vowel. For example, press ^e to produced ê. This is available on a, e, i, o, u.
