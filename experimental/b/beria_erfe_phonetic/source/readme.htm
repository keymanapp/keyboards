Beria Erfe Phonetic

This is a phonetic-based keyboard designed for the Zaghawa Beria language using the Beria Erfe script. Beria Erfe is not yet in Unicode.

The Beria Erfe script has received provisional codepoint assignments by the Unicode Technical Committee. The script has not yet been formally accepted and approved for a future version of the Unicode Standard. The Kedebideri font, and this keyboard, are using the provisional codepoints, but those codepoints can always change. Should those codepoints change, the font and keyboard would be re-released with the new encoding. Documents would then need converting to use the new codepoints.
