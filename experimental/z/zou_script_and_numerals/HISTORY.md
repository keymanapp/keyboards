# Zou Script and Numerals Change History



## 1.0 (2026-05-06)

* Initial release of the Zou Script and Numerals keyboard.
* Supports 27 Zou characters mapped to the Private Use Area (PUA) range U+E000 - U+E039.
* Includes Zou numerals mapped to the PUA range U+E040 - U+E049.
* Developed by Dr. T Marvin Zou.
* Includes an integrated TrueType Font (TTF) with SIL Open Font License (OFL).
* Features a visual on-screen keyboard (KVK) for desktop and web use.
