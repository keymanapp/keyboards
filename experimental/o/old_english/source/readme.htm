Old English

Thanks for installing the Old English keyboard. This keyboard contains all the characters needed to write in Old English and Middle English, including long-s, thorn, that (eth), wynn, yogh, vowels with macrons, various diacritics, insular and dotted characters, Trionian "and", and Middle Welsh "Ll".

To learn Old English, watch Leornende Eald Englisc: https://youtube.com/@leornendeealdenglsc

To learn how to write in runes, watch Rune Revival: https://youtube.com/@runerevival
