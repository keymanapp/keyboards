Hindi Flick keyboard
==============

Description
-----------
Hindi Flick is a mobile keyboard layout based upon the Devanagari varnamala's (alphabet) clustering of syllables. It is intended to be usable by anyone who knows the varnamala. It is inspired by the flick-based layouts popular in Japan and (to a lesser extent) Korea, as well as Vietnamese TELEX with regards to diacritic rules and magic.

हिंदी फ्लिक एक देवनागरी वर्णमाला के अक्षरों के वर्गीकरण पर आधारित मोबाइल कुंजीपटल लेआउट है। इसका उद्देश्य है कि यह किसी भी वर्णमाला के जानकार के द्वारा उपयोगनीय हो। यह जापान और (कुछ कम स्तर तक) कोरिया में प्रसिद्ध "फ्लिक"-आधारित कुंजीपटल, तथा वियतनामी टेलेक्स में पाय जाने वाले मात्राओं को लेकर नियम से प्रेरणा लेता है।
