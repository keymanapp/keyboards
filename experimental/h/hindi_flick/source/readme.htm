Hindi Flick

A mobile keyboard layout designed for Hindi, utilizing a flick input method.

Hindi Flick is a mobile keyboard layout based on the Varnamala, where each key corresponds to a set of syllables. It is inspired by the flick-based layout popular in Japan as well as Vietnamese TELEX with regards to diacritic rules.

The layout is as follows:


There are 16 total keys, with 12 in the center being swipeable in multiple directions, longpress-able and multitap-able.
