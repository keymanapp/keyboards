Àlkèláŋg keyboard
===================

Copyright (C) NIBANN ENGINEERING

__DESCRIPTION__

This keyboard layout seeks to follow the General Alphabet of African Languages adapated for mobile and tablet devices.

Details
--------
- License : MIT
- Author : SOSA Developments
- Platforms: Mobile, Tablet, Android, iOS
- Version : 1.0

Features
--------
- Full support for African letters and diacritics
- Smart long-press functionality
- Dictionaries for word suggestions
- Quick access to the different keyboards: ABC, IPA, Symbols and Currencies.
- Compatible with Keyman 18.0+

Supported Platforms
-------------------
 * Mobile
 * Tablet
 * iOS
 * Android

Installation
------------
1. Compile : `kmc build .`
2. Install the file `build/alkelang.kmp` in the Keyman Mobile app.
3. Test the keyboard.
