Tham Lanna (Thai Layout) keyboard
==============

Description
-----------
The Tham Lanna (Thai Layout) keyboard enables input of the Tai Tham Unicode script using a Thai-based layout. By Jayasiri Jayasāro Bhikkhu, ven.chaisiri@gmail.com.
