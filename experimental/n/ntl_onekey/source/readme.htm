NTL OneKey Keyboard
新傣文一键键盘

Introduction 简介

This keyboard was developed to input New Tai Lue, a script which is used for the Tai Lue (aka Tai Lü, Tai Le, Dai Lue, Xishuangbanna Dai, etc.) language in Xishuangbanna Prefecture in Yunnan, China. The NTL OneKey keyboard is designed to work with the Nokyung font, but you can use this keyboard with any Unicode New Tai Lue font.
