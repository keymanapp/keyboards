Shanghai Telex Change History
====================

1.1 (2026-04-29)
----------------
* updated by cathaylab
* added letter "ŏ" to replace "au"
* added consonant "đr" to replace "dž"
* added ä vowel for [æ]
* added ö vowel for [ɤ]
* deleted ŭ letter and adopted "ư" to represent [ɿ].
* added ü for [ʮ] to replace to former "ư" at this phoneme.
* added consonant "ě" to replace "iê"
* added ą ą̆ ą̈ ę ę̂ į ǫ ǫ̂ ų ų̛  ưǫ̛  nasal vowels to replace ⁿ nasal mark.
