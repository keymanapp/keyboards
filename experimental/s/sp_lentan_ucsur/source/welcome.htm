Sitelen Pona (Lentan, UCSUR)

This is a keyboard for typing sitelen pona, a logographic writing system for the popular constructed language toki pona. This keyboard uses the spelling of words in Latin script to enter sitelen pona logographs. On desktop, letters not in toki pona will enter a syllable instead.

This keyboard is based on the layout described by jan Lentan which was inspired by the toki pona ASCII syllabary by jan Misali.

The sitelen pona logographs produced by this keyboard use the UCSUR encoding. A sitelen pona font supporting this encoding is required for text to appear correctly. (The Keyman mobile app uses a font included in this package.)

Mobile Layout

The mobile layout contains keys only for Latin letters needed in toki pona. When the space bar or enter key is pressed, a toki pona word written in Latin script will be converted to sitelen pona.

The characters shown in grey can be accessed by holding down the corresponding key. Most of these characters are not used in toki pona.

The top left and top right keys input the start and end of a cartouche, respectively. Brackets and extended glyphs can be accessed by holding down these keys. The bottom right key (between space and enter) inputs an interpunct or colon. Combining glyphs can be accessed by holding down this key.

Characters not used in toki pona can be entered using Option on Mac OS X, Right Alt on Linux and Windows. This will change the layout into a standard ASCII keyboard layout.
