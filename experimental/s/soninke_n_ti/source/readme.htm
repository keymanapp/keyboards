Sooninkanxanne N'ti / Soninke N'ti

هذه لوحة مفاتيح لكتابة لغة السوننكي في نص N'ti. لم يتم ترميز البرنامج النصي N'ti بعد في الترميز الموحد / Il s'agit d'un clavier permettant d'écrire la langue Soninké en écriture N'ti. L'écriture N'ti n'est pas encore codée en Unicode. / This is a keyboard to write the Soninke language in the N'ti script. The N'ti script is not yet encoded in Unicode.
