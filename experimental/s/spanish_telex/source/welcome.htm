Spanish Telex for Spanish

Spanish Telex keyboard is a keyboard uses key combination method to type Spanish characters with marks and special punctuations.

Character Key pressed
á aa

To write the pair of keys as two distinct characters, the second character has to be repeated. For example, á + a > aa, ¿ + ? > ??
