Punjabi Flick keyboard
==============

Description
-----------
The Punjabi Flick Keyboard is an innovative keyboard designed primarily for mobile devices, enabling users to efficiently input Punjabi letters with speed and precision. Inspired by the [Hindi Flick Keyboard](https://keyman.com/keyboards/hindi_flick), as seen on Keyman. (A little too much)
