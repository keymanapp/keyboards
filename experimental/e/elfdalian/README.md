Elfdalian keyboard
==============

©2023

Version 1.05

Description
-----------

This keyboard is designed for typing the Elfdalian language in the Latin script. 
Some fonts fail to display the character "̨Y" correctly. One font which reliably displays it correctly is Consolas.
