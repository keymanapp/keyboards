Start Using Finongan

Finongan keyboard based on US. In order to add non-Finongan characters (j q r v x z), type the semi-colon [;] and then the desired letter.

Stat Long Usim Finongan Kibod

I usim kibod bilong US long wokimg Finongan kibod. Sapos yu laik usim ol leta i no stap long Finonga alphabet, yu mas taipim ; bipo yu taipim dispela leta yu gat laik long usim.
