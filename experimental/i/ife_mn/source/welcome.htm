Start Using Ifè Mnemonic

Keyboard for the Ifè language of Togo.

Keyboard Layout

Use "`" followed by vowel to access grave accent.

Use "'" followed by vowel to access acute accent.

Use ";" followed by vowel to access tilde accent.

Use "^" followed by vowel to access circumflex accent.

Use ":" followed by vowel to access diaeresis accent.

Use "~" followed by vowel to access tilde+grave accent.

Use '"' followed by vowel to access tilde+acute accent.

";" is also used in other sequences to access special characters. For example ";n" for eng, ";?" for inverted question mark, etc.
