Start Using Ishkashimi

Ishkashimi Cyrillic Keyboard.

For the letters combined with diacritic symbols, type first the letter, then type the diacritic symbol. The diacritic symbols are on the top row under the keys 1, 2 and 3.

The fonts that work best with Ishkashimi letters are Cambria, Calibri, Noto Sans.
