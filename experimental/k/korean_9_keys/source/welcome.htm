Keyboard Layout

The old hangul keyboard layout in computer is similar with ordinary korean keyboard in computer, with additional keys for old hangul letter.

The touch-layout is similar with 9 keys korean keyboard in the phone, with additional flick keys for old hangul letters.

Using flick to type different letters

Jamo Generating

The generating rules for old hangul is similar with typing with normal jamo keyboard. You can combine each jamo letters or jamo strokes to generate combined consonants, combined vowels, and jamo syllable. The generating happens from left to right and from up to down.

For example:

ㅣ + ㅇ > ㆁ

ㄹ + ㅇ > ᄛ

ㅂ + ㅇ > ㅸ

ㅍ + ㅇ > ㆄ

ㅕ + ㅣ > ㅖ

ㅑ + ㅣ > ㅒ

얘 + ㅣ > ᅌퟀ

이 + ㅣ > ᄋퟄ

Typing space while pressing shift to end generating of one syllable for start next syllable.

Adding a space then deleting the space will have the same effect.
