Tawrã-Kaman Keyboard

This keyboard is designed for the Tawra (Digaro Mishmi, Taraon) and Kaman (Miju Mishmi) languages.

Key Features:

  • Smart-Transform: Tones and nasalization marks automatically combine with vowels.
  • Mobile Optimization: Includes Auto-Shift and Double-tap Caps Lock.
  • Full Punctuation: Standard symbols available via double-tap or Shift.

Based on Keyman multilingual keyboard system (https://keyman.com/), developed by NEILAC, Guwahati (2026). Thanks to the Keyman Team for their support.
