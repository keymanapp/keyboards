The GFF Tigrinya Keyboard Package

The contents of this package contain both free (OpenSource) and commercial software from various organizations. Please review the following for a complete list of the provided software components and documentation governing their use and availability.

Keyboards

Two keyboards are provided with this package for Ethiopic script under the conventions of the Tigrinya language, and the full Ethiopic in a language neutral form. The keyboards are free software and you may share them freely with friends. The keyboards themselves may be downloaded independently from ftp://ftp.ethiopic.org/pub/keyboards/Keyman/7.0/.

The software required to run the keyboards, "Keyman" is a commercial product developed by the Tavultesoft Proprietary Limited Company. Keyman is free for a period of time before it must be purchased from Tavultesoft to continue functioning. See the Tavultesoft homepage for purchasing details.

The Tigrinya keyboard is "mnemonic" and designed for the US English "QWERTY" keyboard. This means that the keyboard is designed to be intuitive and natural with respect to the sounds available in the English language via the standard English keyboard (known as "QWERTY"). The package also contains some extensions to work mnemonically on non-English European keyboards.

Fonts

The Abyssinica SIL Font

The SIL International is responsible for the Abyssinica SIL font which is included with this package and is licensed under the Open Font License (OFL). The Abyssinica SIL font is a high quality font which includes all Ethiopic letters and symbols in the Unicode 4.1 standard.

The Abyssinica SIL font is automatically installed into your "Fonts" folder and has the file name Abyssinica_SIL.ttf and the font menu name "Abyssinica SIL".

See the Abyssinica SIL Homepage

The Free Serif Font

The GNU FreeFont Project is responsible for the Free Serif font which is included with this package and is licensed under the General Public License (GPL). The Abyssinica SIL font is a modest quality font which includes all Ethiopic letters and symbols in the Unicode 3.0 standard.

The Free Serif font is automatically installed into your "Fonts" folder and has the file name FreeSerif.otf and the font menu name "Free Serif".

See the GNU FreeFont Project Homepage

The WashRA Font Collection

The Senamirmir Project is responsible for the WashRA fonts which are included with this package and is licensed under the General Public License (GPL). The WashRA collection are a high quality fonts which includes all Ethiopic letters and symbols in the Unicode 3.0 standard.

The WashRA fonts are automatically installed into your "Fonts" folder and have the file and menu names:

  • fantuwua.ttf and the font menu name "Ethiopic Fantuwua".
  • goffer.ttf and the font menu name "Ethiopic Yigezu Bisrat Goffer".
  • hiwua.ttf and the font menu name "Ethiopic Hiwua".
  • jiret.ttf and the font menu name "Ethiopic Jiret".
  • jiretsl.ttf and the font menu name "Ethiopic Jiret Slant".
  • tint.ttf and the font menu name "Ethiopic Tint".
  • washrab.ttf and the font menu name "Ethiopic WashRa Bold".
  • washrabsl.ttf and the font menu name "Ethiopic WashRa Bold Slant".
  • washrasb.ttf and the font menu name "Ethiopic WashRa SemiBold".
  • washrasbsl.ttf and the font menu name "Ethiopic WashRa SemiBold Slant".
  • wookianos.ttf and the font menu name "Ethiopic Wookianos".
  • yebse.ttf and the font menu name "Ethiopic Yebse".

See the WashRA Homepage
