This is a phonetic/alphabetic keyboard. Typing k gets you क, and कमला is keyed in as kamalaa or kamalA. Gautami Devanagari deviates from customary Devanagari orthographic practice of omitting the inherent अ/a/schwa in a consonant.

You *must* read the document gautami_devanagari.pdf before you start using this keyboard.
