This is a phonetic/alphabetic keyboard. It requires the inherent অ/schwa in a consonant to be explicitly typed in, e.g কি is typed in as ki, but কই must be typed in as kai. Use LeftCtrl + LeftShift + B to turn this keyboard on or off, and LeftAlt + LeftShift + K to invoke or revoke an on-screen keymap.

You *must* read the document gautami_bangla_bengali.pdf before you start using this keyboard.
