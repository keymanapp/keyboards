This is an NOT a phonetic or alphabetic keyboard. It follows the traditional Brāhmī akṣara-based orthographic practice of omitting the inherent அ/schwa in a consonant, e.g பகடி is keyed in as pkTi, not pakaTi. Use LeftCtrl + LeftShift + T to turn this keyboard on or off, and LeftAlt + LeftShift + K to invoke or revoke an on-screen keymap.

Download the document gautami_tamizh_tamil.pdf and study it carefully before you start using this keyboard.
