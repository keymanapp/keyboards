This is a phonetic keyboard: typing k gets you க. However, it is not alphabetic. It follows the Brāhmī orthographic practice of omitting the inherent அ/schwa in a consonant, e.g பகடி is keyed in as pkTi, not pakaTi. Use LeftCtrl + LeftShift + T to turn this keyboard on or off, and LeftAlt + LeftShift + K to invoke or revoke an on-screen keymap.
Please read the document gautami_thamizh_tamil.pdf carefully before you start using this keyboard.
