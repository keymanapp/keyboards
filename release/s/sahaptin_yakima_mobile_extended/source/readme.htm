Sahaptin/Ichishkíin (Yakima) Keyboard [Mobile Expanded]

Keyboard (c) 2019-2020 || Keegan Livermore

This project provides a version of the mobile keyboard for both Android and iOS that features a full layout with all characters visible with "alternate" versions on key-hold options. This is recommended for those either more familiar with smaller buttons or with a larger screen, on a phone or a tablet.
