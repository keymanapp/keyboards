Start Using Sorani Behdini (Qwerty)

This keyboard is designed for typing Kurdish Sorani Behdini in the Arabic script. It is based on a phonetic layout.

There are a few keys which are only available through two keystrokes:

  • RALT+COMMA produces ›, and pressing RALT+COMMA a second time will produce »
  • RALT+PERIOD produces ‹, and pressing RALT+PERIOD a second time will produce «
  • RALT+SHIFT+COMMA produces ’, and pressing RALT+SHIFT+COMMA a second time will produce ”
  • RALT+SHIFT+PERIOD produces ‘, and pressing RALT+SHIFT+PERIOD a second time will produce “

Mobile Keyboard layout
