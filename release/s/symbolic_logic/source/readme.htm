Symbolic Logic

© Christopher V. Mirus

The Symbolic Logic keyboard for Keyman makes it possible to type logical formulae without learning LaTeX or inserting special characters from a menu. The Shift and Alt keys give access to:

·         the complete Latin and Greek alphabets

·         the most common symbols of logic and arithmetic

·         the symbols of set theory most commonly used in logic

·         a handful of less common logical symbols

·         superscripted numerals for indicating the adicity of predicates
