Start Using siddham_inscript

The Siddham script is a descendant of the Brahmi script originating in India. It was used to write Sanskrit during 3rd to 6th century AD and spreaded along the Silk Road. It is still used now in China, Japan and other regions in eastern Asia.
This keyboard is designed for the Sanskrit language, using the Siddham script with an Inscript layout.
