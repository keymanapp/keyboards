Starlingŝava klavarfasono 1.0

Ĉi tiu klavarfasono ebligas al vi, skribi esperante per la alfabeto starlingŝava. Tiuj ĉi literoj estos sur la unua tavolo (alivorte, sur la "minuskloj" de via klavaro). La klavaro ankaŭ inkludas latinajn diakritaĵojn, kiuj estis plilongigitaj por skribi fremdajn lingvojn.

Komputile: se la klavo ⇪ CAPS Lock estas ŝaltita, vi povos skribi esperant-latinajn minusklojn; ⇧ SHIFT ŝaltita faros majusklojn.
