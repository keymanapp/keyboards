Naskapi (Common) Keyboard
=====================

Copyright (c) 2001-2020 Bill Jancewicz

Version 1.4

__DESCRIPTION__
The Naskapi (Common) keyboard is a phonetic-style 
(type the word the way it sounds) layout. It uses Unified Canadian Aboriginal Syllabics.
