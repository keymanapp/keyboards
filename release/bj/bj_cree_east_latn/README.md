Hâts (east) keyboard
==============

Description
-----------

Roman keyboard for typing vowel length (circumflex "hats" over vowels) in roman orthography (East Cree).

-------------------

 * macOS
 * Linux
 * Web
 * iPhone
 * iPad
 * Android phone
 * Android tablet
 * Mobile devices
 * Desktop devices
 * Tablet devices
