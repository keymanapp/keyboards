Start Using Maltese

This keyboard is designed for the Maltese language.

Vowels
'a 'e 'i 'o 'u 'y   →   á é í ó ú ý
"a "e "i "o "u "y   →   ä ë ï ö ü ÿ
`a `e `i `o `u     →   à è ì ò ù  
^a ^e ^i ^o ^u     →   â ê î ô û  
 
Consonants
'c 'g 'h 'z       →   ċ ġ ħ ż    

Press an accent key twice to output the accent.
