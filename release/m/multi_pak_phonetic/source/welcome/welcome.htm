Multi Language Pakistan Phonetic Keyboard

Created by Zeeshan Hyder

This package includes a phonetic keyboard called Multi Pak Phonetic designed for the following languages: Balochi, Kashmiri, Memoni, Panjabi, Pashto, Seraiki, Sindhi and Urdu. Please note that the On Screen Keyboard is based on a US English keyboard layout.
