Start Using Me’en (Latin)

Me’en (Latin) keyboard based on US keyboard layout.

Keyboard Layout

Unshifted

To get à (low tone a) please type a` To get á (high tone a) please type a``

To get ɛ̀ (low tone ɛ) please type ɛ` To get ɛ́ (high tone ɛ) please type ɛ``

and so on...

The various quote characters are typed by pressing the ’ key, which is also used for the consonant character ’ (appearing in the word Mɛ’ɛn for example).

So to get the various single quote characters please press ’ once, twice, or three times respectively.

To get the various double quote characters please press SHIFT ’ once, twice, three times respectively.

(For realizing straight quotes (single or double) please ensure to adapt autotype/autocorrect settings accordingly in your word processor.)

Shifted

To get À (low tone capital a) please type A` To get Á (hight tone capital a) please type A``

To get Ɛ̀ (low tone capital ɛ) please type Ɛ` To get Ɛ́ (hight tone capital ɛ) please type Ɛ``

and so on...

In order to get € please press the combination for $ twice.

In order to get £ please press the combination for $ three times.
