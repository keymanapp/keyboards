Start Using miluk_hanis_siuslaw

This keyboards were created by Jordan AG Douglas-Tavani for typing Hanis Kuukwis (Hanis), Miluk Kuukwis (Miluk), and Shaʼyuushtlʼa Quuiich (Siuslaw) in accordance with the Confederated Tribes of Coos, Lower Umpqua, and Siuslaw Indians classroom orthography. The computer-based keyboard does not represent every sound in the languages, but rather, the characters used to create those sounds. This means that the 'ch' sound is created by combining both the characters of 'c' and 'h'. Meanwhile, the mobile keyboards have longpress utilization. This allows for all sounds in the language to be represented, such as 'ch', 'sh', 'hw', etc.; with their own keys. If you have any questions, comments, or concerns or if you need any help, please reach out to jdouglas@ucsb.edu.
