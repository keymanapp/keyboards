Este es el teclado de Keyman mayan_us
Está diseñado para ser utilizado con Keyman 9 o superior y fue creado para los idiomas de Guatemala.

El uso básico del teclado es el siguiente:

[ + vocal = vocal acentuada
= + vocal = vocal subrayada (vocal larga)

; + [n s u x z] = [n̈ s̈ ü ẍ z̈]

= + = produce el guion largo

< + < = “
“ + < = ‘
> + > = ”
” + > = ’
