This is the mayan_us Keyman keyboard
It is designed to be used with Keyman 9 or higher and was created for the languages of Guatemala.

The basic use of the keyboard is as follows:

[ + vowel = accented vowel
= + vowel = underlined vowel (long vowel)

; + [n s u x z] = [n̈ s̈ ü ẍ z̈]

= + = produces the “em-dash”

< + < = “
“ + < = ‘
> + > = ”
” + > = ’
