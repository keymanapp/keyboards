Start Using Manichaean

This keyboard is for languages using the Manichaean script.

Standardized Variation Sequences are accessed by typing "d", "h", "m", "n", or "r", followed by "=" which is U+FE00.

To type numbers: 1 + 0 becomes 10, 1 + 0 + 0 becomes 100.
