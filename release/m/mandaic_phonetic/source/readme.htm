Mandaic Phonetic

This keyboard is designed for typing the Mandaic language in the Mandaic script. A Mandaic font is required for displaying the characters. This keyboard package will install the Noto Sans Mandaic font.

Mandaic is a right-to-left script. In order to use the script in Microsoft Office or LibreOffice, right-to-left behavior must be enabled. This page descibes how to enable right-to-left in several applications: Enabling right-to-left in various applications.
