၁။ Keyboard Layout ၂။ ဗီုဂွံတက်ဗျန် ၃။ အက္ခရ်တၟေၚ် ၄။ လၟေင်သွဵု

ပ္ဍဲအရာမတက်အက္ခရ် Mon Unicode Keyboard ဝွံ တၟေင်တၞဟ်ခြာဟွံမွဲရ၊ လၟေင်တက်အက္ခရ်မပ္တံကဵု ၜ၊ဂ၊ယ၊ဣ ကွက်နတအ်ဂှ် တၟေင်နွံတဴညိညရ

ဥပမာ = တက်အက္ခရ် ၜ မ္ဂး \ , ဂ shift+:, ယ shift+B, ဣ shift+M တက်ဂစိုတ်ကင်မ္ဂး ဍေံတိတ်ဂစိုတ်ကင် ဟွံမွဲကဵုကွက် ၚ အလဵုအလဵု Auto fix system ရ။

မပ္တံကဵု Key ဗၟာတအ်လေဝ် တၞဟ်နဂွံသုင်စောဲ ဗွဲမဂတာပ်အခိင်ကီု တၞဟ်နဟွံဒးပြံင် Keyboard ဗၟာကီုတုဲ၊ မ္ဒးထပ်စုတ်လဝ် သီု Key ဗၟာရ။ ယဝ်မိက်ဂွံတက် Myanmar Key မ္ဂး ဍဵု Ctrl + Alt ဥပမာ : တက် "ဦ" မ္ဂး Ctrl + Alf + M, ဦ y, ၌ [,ဧ
