#⚙️ Installing the Mara Keyboard

This guide explains how to install and use the **Mara Keyboard** across different devices using the **Keyman** platform.

---

## 🖥️ Desktop Installation (Windows, macOS, Linux)

### Step 1: Install Keyman
Download and install the **Keyman Desktop** application:  
👉 [https://keyman.com/downloads](https://keyman.com/downloads)

### Step 2: Install the Mara Keyboard
After installing Keyman, choose one of the following:
- **Option A:** Open Keyman → click **“Add Keyboard”** → search for **“Mara”**.  
- **Option B:** Go directly to [https://keyman.com/keyboards/mara](https://keyman.com/keyboards/mara) and click **“Install Keyboard”**.

### Step 3: Enable the Keyboard
- Open the Keyman menu in your taskbar or menu bar.  
- Select **Mara Keyboard** as your active layout.  
- Start typing Mara text in any Unicode-compatible application.

---

## 📱 Mobile Installation (Android & iOS)

### Step 1: Install the Keyman App
- **Android:** [Google Play Store – Keyman](https://play.google.com/store/apps/details?id=com.tavultesoft.kmapro)  
- **iOS:** [App Store – Keyman](https://apps.apple.com/app/keyman/id993722845)

### Step 2: Add the Mara Keyboard
1. Open the **Keyman App**.  
2. Tap **“Add Keyboard” → “Download Keyboard”**.  
3. Search for **“Mara”** and select **Install**.

### Step 3: Start Typing
You can now type Mara text inside the Keyman app or enable the keyboard for system-wide use from your phone’s **keyboard settings**.

---

## 💡 Tips

- Switch between English and Mara using the **🌐 Globe key**.  
- Long-press keys for additional tone marks or diacritics.  
- Refer to the [Keyboard Help Page](https://help.keyman.com/keyboard/mara/1.0.1/mara) for key layout charts.

---

## 🧹 Uninstallation

If you need to remove the keyboard:
- **Desktop:** Keyman → Configuration → Installed Keyboards → Remove.  
- **Mobile:** Keyman App → Installed Keyboards → Delete.
