# 🕒 Mara Keyboard – Version History

The changelog for the **Mara Keyboard**, documenting all notable updates and improvements.

---

## 🧩 Version 1.0.1
**Release Date:** February 14, 2022  
**Documentation:** [https://help.keyman.com/keyboard/mara/1.0.1/mara](https://help.keyman.com/keyboard/mara/1.0.1/mara)

### 🆕 Changes
- First official stable release on the Keyman platform.  
- Added full support for **desktop and mobile layouts**.  
- Optimized Unicode mapping for Mara characters.  
- Improved consistency across tone marks and vowel signs.  
- Compatible with **Keyman 15+**.

---

## 🧪 Version 1.0.0 (Beta)
**Release Date:** December 10, 2021

### 🔧 Changes
- Beta version released for community testing.  
- Included base layout for Mara script.  
- Gathered user feedback to refine diacritic handling and usability.

---

## 🔮 Planned Updates
- Optimize touch layout for smaller screens.  
- Enhance tone mark rendering and keyboard spacing.  
- Add additional documentation and visual layout charts.
