# 📝 Mara Keyboard for Keyman

The **Mara Keyboard** enables easy and accurate typing in the **Mara language**, used primarily in Myanmar and surrounding regions.  
Built for the **Keyman platform**, it supports both desktop and mobile devices to make Mara writing more accessible and consistent.

---

## 🌐 Overview

| Property | Details |
|-----------|----------|
| **Keyboard Name** | Mara Keyboard |
| **Language** | Mara (ISO 639-3: `mrh`) |
| **Platforms** | Windows, macOS, Linux, Android, iOS, Web |
| **Maintainer** | [Keyman](https://keyman.com/keyboards/mara) |

This keyboard layout provides a convenient and intuitive input system for Mara Unicode characters.  
It has been developed with community feedback to ensure accurate tone marks, diacritics, and smooth usability across devices.
