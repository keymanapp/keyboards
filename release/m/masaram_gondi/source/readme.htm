Masaram Gondi

Description

This is a Phonetic (ITRANS) keyboard layout for Gondi languages written in Masaram Gondi script. It uses the ITRANS transliteration scheme by Avinash Chopde, with some minor modifications for typing phonetically using a QWERTY keyboard and getting output in Masaram Gondi script in Unicode.
