Mozhi Malayalam Keyboard

Mozhi is arguably the most popular phonetic/transliteration based keyboard layout for Malayalam script. This keyboard was designed to mimic the Mozhi keyboard spec and updated to include all the recent additions of the characters (mostly archaic) to the Malayalam Unicode Block. This keyboard is designed to type Malayalam using an unshifted and a shifted keyboard. So no complex key combinations are required to keyboad the text. Most Malayalam words can be keyboarded with the intuitive keys as it try to mimic the way most Malayalam speakers transliterate Malayalam using English alphabets. The keyboard types in a phonetic style based on the English (QWERTY) layout. This keyboard currently supports Unicode 10.


Fonts

This keyboard comes packaged with the AnjaliOldLipi.ttf font. However there are several other free and open Malayalam fonts can be downloaded from the following places.

  • Anjali Fonts (Both Old and New Orthography)
  • SMC Fonts
  • Aruna Font
