Mozhi Malayalam Keyboard

This keyboard is designed to type Malayalam using an unshifted and a shifted keyboard. The keyboard types in a phonetic style based on the English (QWERTY) layout. This keyboard currently supports Unicode 5.1.


Fonts

This keyboard comes packaged with the AnjaliOldLipi.ttf font,. To find other fonts on your computer which support this keyboard, use the Keyman Desktop Font Helper.
