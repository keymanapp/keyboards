Mozhi Malayalam Keyman Keyboard Updated 2.0

The first implementation of the Mozhi Malayalam keyboard was done by Raj Nair (Peringodan) in 2015, but the source for the same is lost. This version is a complete re-write, and is updated to match the current spec of Mozhi Scheme Version 2.0 (Unicode 11)

Mozhi is aguably the most popular keyboard layout used by regular netisons for keyboarding Malayalam script. Its wide acceptance is primarilty due to its intuitive design, which match very close to the way people write Malayalam using the Latin alphabets. So this is mainly a transliteration type scheme and it was originally developed for Varamozhi Application.
