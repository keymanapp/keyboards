Mozhi Malayalam Keyman Keybaord Updated 2.0

The first implementation of the Mozhi Malayalam keyboard was done by Raj Nair (Peringodan) in 2015, but the source for the same is lost. This version is a complete re-write, and is updated to match the current spec of Mozhi Scheme Version 2.0 (Unicode 11)

Mozhi is aguably the most popular keyboard layout used by regular netisons for keyboarding Malayalam script. It's wide acceptance is primarilty due to its intuitive design, which match very close to the way people write Malayalam using the Latin alphabets. So this is mainly a transliteration type scheme and it was originally developed for Varamozhi Application.

അക്ഷര വിന്യാസവും (layout) സ്വാഭാവിക മലയാള എഴുത്തും മൊഴി 2.0-ൽ

സാധാരണക്കാരായ മലയാളികൾക്ക് ഏറ്റവും സുപരിചിതമായ മലയാളം റ്റൈപ്പിംഗ് ലേയൗട്ട് മൊഴിയായിരിക്കും.

ഡിജിറ്റൽ മലയാളത്തിൻ്റെ വളർച്ചയ്ക്കനുസരിച്ച് ഉണ്ടായ കാലാനുസൃതമായ മാറ്റങ്ങളോടെ മൊഴിയുടെ പുതുക്കിയ പതിപ്പ് ഇപ്പോൾ തയ്യാറായിരിക്കുകയാണ്. ഇത് ഓൺലൈനിൽ കാണുവാനായി ഈ ലിങ്ക് സന്ദർശിക്കുക. യൂനിക്കോഡ് 11 അനുസരിച്ചുള്ള എല്ലാ മലയാള അക്ഷരങ്ങളും ഈ പതിപ്പിൽ ലഭ്യമാണ്.

അക്ഷരമാപനം/വിന്യാസം (KEY MAPPING)

മൊഴിയിലെ അക്ഷരവിന്യാസം മലയാളികൾ മംഗ്ലീഷിൽ റ്റൈപ്പ് ചെയ്യുന്ന പൊതുവായ രീതിയോട് വളരെ സാമ്യമുള്ളവയാണ്. ഉദാ: ‘അമ്മ’ എന്നതിനു ‘amma’ എന്നു റ്റൈപ്പ് ചെയ്താൽ മതി. ഇതുപോലെ സാധാരണങ്ങളായ പല പദങ്ങളും മംഗ്ലീഷിൽ എഴുതിയാൽ അവയുടെ മലയാളം നേരെ ലഭിക്കുന്നതാണ്.

താഴെക്കൊടുത്തിരിക്കുന്ന ഉദാഹരണങ്ങൾ കൂടി നോക്കുക:

  • pappa → പപ്പ
  • mathi → മതി
  • nee/nI → നീ
  • avan → അവൻ
  • kuttikkaalam → കുട്ടിക്കാലം
  • nakshathram → നക്ഷത്രം
  • vaazhappazham → വാഴപ്പഴം

എന്നാൽ ഇത് എല്ലായ്‌പ്പോഴും അങ്ങനെയാകണമെന്നുമില്ല. ഉദാ: വട എന്നത് പലപ്പോഴും vada എന്നാണ് മംഗ്ലീഷിൽ എഴുതുന്നത്. മൊഴിയുപയോഗിച്ച് vada എന്നെഴുതിയാൽ വദ-യെന്നേ വരൂ. ഇതിനു പല കാരണങ്ങൾ ഉണ്ട്.

  • പലപ്പോഴും നമ്മൾ എഴുതുന്നത് ഒരു രീതിയിലും പറയുന്നത് ഒരല്പം മാറ്റിയുമായതുകൊണ്ടാണ്. (ഇതിൻ്റെ പുറകിലെ ഭാഷാശാസ്ത്രം വിശദീകരിക്കുന്നത് ഈ പോസ്റ്റിൻ്റെ ലക്ഷ്യത്തിനു പുറത്തായതിനാൽ അതിനിവിടെ മുതിരുന്നില്ല).
  • മംഗ്ലീഷിൽ പലപ്പോഴും ഒരേ ഇംഗ്ലീഷ് അക്ഷരം കൊണ്ട് ഒന്നിലധികം മലയാള അക്ഷരങ്ങളെ കുറിക്കാറുണ്ട്. ഉദാ: paatta എന്നത് പാട്ടയ്ക്കും പാറ്റയ്ക്കും ഉപയോഗിക്കുന്നത്.
  • സാഹചര്യങ്ങൾ അനുസരിച്ച് ചേർത്തോ പിരിച്ചോ എഴുതേണ്ടി വരിക. ഉദാ: vanyavanika → വന്യവനികയോ (wild creeper) വൻയവനികയോ (big curtain) ആകാം
  • മറ്റൊന്ന് മംഗ്ലീഷ് എന്നത് ഏകീകരിക്കപ്പെട്ടിട്ടുള്ള (Standardized) എഴുത്ത് രീതിയൊന്നുമല്ല. ഒരാൾ മംഗ്ലീഷിൽ എഴുതുന്നതുപോലെയാകണം മറ്റൊരാൾ എഴുതുന്നത് എന്ന് നിർബന്ധമൊന്നുമില്ല. ഉദാ: വിജ്ഞാനം എന്നെഴുതാൻ ചിലർ vignjanam എന്നെഴുതും, പക്ഷേ മറ്റു ചിലർ vijnanam എന്നായിരിക്കും എഴുതുക.

അതുകൊണ്ട് ചിലപ്പോഴെങ്കിലും നമ്മൾ ഉദ്ദേശിക്കുന്ന അക്ഷരങ്ങൾ നേരിട്ടു കിട്ടാതെ വരാൻ സാധ്യതയുണ്ട്.

മൊഴിയിൽ ഇത്തരം സാധ്യതകൾ മുമ്പിൽ കണ്ടുകൊണ്ട് പല മലയാള അക്ഷരങ്ങളും ഒന്നിലധികം രീതിയിൽ റ്റൈപ്പ് ചെയ്യാൻ ഉള്ള മാർഗങ്ങൾ ഉൾപ്പെടുത്തിയിട്ടുണ്ട്. ഈ ഉദാഹരണങ്ങൾ കാണുക:

  • പാടം ← paaTam / pATam
  • വണ്ടി ← vanTi / vaNTi / vaNti
  • കൂടുതൽ ← kUTuthal / kooTuthal / kuuTuthal
  • അച്ഛൻ ← achchhan / aChan
  • ക്ഷണം ← kshaNam / Xhanam

ഇത്രയൊക്കെയാണെങ്കിലും മൊഴി നന്നായുപയോഗിക്കാൻ അല്പം പരിശീലനം കൂടിയേ തീരൂ. അതിനായി മൊഴിയുടെ ചില അടിസ്ഥാന തത്വങ്ങളെ മനസ്സിലാക്കണം.
