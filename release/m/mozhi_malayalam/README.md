This keyboard uses the Mozhi layout to let you type in Malayalam. It is a phonetic keyboard, based on the normal English (QWERTY) keyboard letters. This keyboard supports Unicode 10.

Supported Platforms
-------------------
 * Windows
 * Linux
 * macOS
 * Web
