Mongolian Cyrillic QWERTY Keyboard
=====================

Version 1.0

__DESCRIPTION__
This phonetic keyboard layout is designed for Khalkha Mongolian Cyrillic. 
It is based on the English QWERTY keyboard and also aligns with Chinese Pinyin.

Supported Platforms
-------------------
 * Windows
 * macOS
 * Web
 * Mobile Web
 * iOS
 * Android
