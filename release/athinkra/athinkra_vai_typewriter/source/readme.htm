Vai Typewriter (Athinkra)

A Keyman keyboard for the Vai language of Liberia, developed by the team at Athinkra, LLC. This keyboard uses fewer keystrokes than the "Vai (Athinkra)" keyboard and is more ergonomic. The Dukor and Wakor fonts are a part of the keyboard package and will be installed with the keyboard.
