Pukapuka (pkp) keyboard for Keyman Desktop

Mark Penny
SIL International

This Pukapuka Keyman keyboard allows a user to type the macrons, acutes, graves and circumflexes needed to type the Pukapuka language spoken in the Cook Islands of the south Pacific.

Keyboard Layout

Vowel diacritics are typed as sequences of an appropriate key (; : ` ^). Details of the keyboard layout may be found in the "welcome.htm" document in the section on Keyboard Layout.

Fonts

SIL International has produced several Unicode font sets that allow for keyboarding the Pukapuka language. These fonts are:

    Charis SIL
    Doulos SIL
