Welcome to the Pukapuka Keyboard

The Pukapuka keyboard has been designed for use on physical keyboards (laptops, etc.) as well as hand-held touch devices (such as smart phones, tablets, etc.)

The touch layout has some additional functionality of providing short-cuts for commonly used combinations of characters. For example, holding down (long-press) a vowel key will show various vowels with the diacritics.

For the physical keyboard, the keys for typing diacritics are shown below:
