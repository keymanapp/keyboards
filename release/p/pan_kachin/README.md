Pan Kachin keyboard
==============

Description
-----------
This is a keyboard that contains all of the characters needed to type 5 major Kachin languages: Jingpho, Lashi, Lhaovo (Maru), Zaiwa, and Rawang. The keyboard produces NFD Unicode.
