Poorna Malayalam Extended Inscript Keyboard

Poorna Malayalam Extended Inscript keyboard is a complete Malayalam unicode character set keyboard. The key mapping of this keyboard is an extended version of the Malayalam Inscript layout. Visit https://poorna.smc.org.in for more information.

Keyboard Layout


View / Save

This keyboard consists of four layers. The above layout is structured as follows:

  • Left Bottom : Unshifted
  • Left Top : Shifted
  • Right Bottom : AltGr (Right Alt)
  • Right Top : AltGr (Right Alt)+Shift

Example

In the case of key d :

  • d → ് (U+0D4D : MALAYALAM SIGN VIRAMA)
  • Shift + d → അ (U+0D05 : MALAYALAM LETTER A)
  • AltGr + d → ഽ (U+0D3D : MALAYALAM SIGN AVAGRAHA)
  • AltGr + Shift + d → ഁ (U+0D01 : MALAYALAM SIGN CANDRABINDU)
