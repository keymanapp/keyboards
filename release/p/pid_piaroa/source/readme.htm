This keyboard defines some keys that are set by default as “hotkeys” when Keyman Desktop for Windows is installed. To use this keyboard, you will want to go to Configuration.../Hotkeys, select the hotkey name, and either change the hotkey or turn it off. Currently “Switch Keyman Desktop Off” uses Shift+Alt+O

Este teclado define unas teclas cuales son teclas de acceso rápido (“hotkeys”) por omisión cuando Keyman Desktop para Windows se instala. Para usar estas teclas, debes navegar a Configuration.../Hotkeys, selecciona el “hotkey”, luego debes cambiar el ”hotkey” o desactivarlo enteramente. Desde versión 1.0.0, “Switch Keyman Desktop Off” (“Desactivar Keyman Desktop”) usa Shift+Alt+O
