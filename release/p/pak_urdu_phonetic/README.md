Pak Urdu Phonetic keyboard
==============

Description
-----------
Pak Urdu Phonetic is a faithful port of mBilalm’s popular Windows-based Pak Urdu Installer layout, featuring 160+ key mappings — one of the most comprehensive Urdu phonetic layouts available.
