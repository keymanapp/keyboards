Start Using Indonesia-Suku Basic Keyboard

Physical Keyboard layout

Main features:

  1. Based on QWERTY that we are all familiar with (US basic layout), but without the X and Q letters (statistically occur in less than 0,1% of all letters in Indonesian languages)
  2. Put e-acute and e-accent in place of X and Q
  3. Put X and Q hidden in Z key (longpress). You can also press 'z' repeatedly to access x and q. 'zz' (z+z) = x, 'zzz' (x+z) = q, 'zzzz' (q+z) = back to z.

Touch Keyboard layout

Touch keyboard features:

  1. Based on the physical keyboard layout
  2. Basic: 3 layers (default, shift, numeric), 4 rows (5 for numeric), 11 keys wide. It means large buttons, ease to type keys.
  3. All the necessary symbols (and nothing else) on the default layer (,.?) punctuations, '-' and '2' for reduplication, (") quotation, and '/()' misc punctuation that we often use. The rest of the basic symbols found in US basic layout are in the shift layer. All the symbols can be found in the numeric layer.

Tips:

  1. Tap 'z' repeatedly to access x and q. 'zz' (z+z) = x, 'zzz' (x+z) = q, 'zzzz' (q+z) = back to z.
  2. Longpress: if you see a dot on the top right of a key, it means there's a longpress option.
  3. The '/()' means that there's 3 keys (longpress) in that key.
