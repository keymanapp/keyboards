Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard

A Phonetic Devanagari (ITRANS transliteration scheme) keyboard layout with support for Vedic Sanskrit (Devanagari Extended and Vedic Extensions Unicode ranges).

Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard is based on the phonetic ITRANS transliteration scheme with some modifications. Refer to the table below for the English letters to type to get Devanagari consonants.

Consonant conjuncts are automatically formed, e.g. k t produces क्त्‌. The inherent a is only applied when followed by space. e.g. k t a or k t space produces क्त. To force an explicit virama at end of word, use ^ e.g. k t ^ space produces क्त्‌ .

A full (live) consonant with visible virama in middle of words can be typed using ^, e.g. k ^ S produces क्‌ष.

Other input sequences

Typing Latin letters

Words in written in Latin letters can be typed by first typing Q at the start of the word, for example: Q H i n d i will type the word Hindi.

Numbers

  1. Press the number keys to type Devanagari digits. for example 9 produces ९.
  2. For typing the Arabic digits, press the number keys and then the backspace key, for example 9 Back space produces 9.

Yajurvedic Combining Marks

  1. For typing the Yajurvedic combining mark with digits, press the number keys and then the equal key, for example 9 = produces ꣩.
  2. For typing the other Yajurvedic combining marks, press the letter key and then the equal key as per the following table.
Key Output Character Comment
a= ꣪ COMBINING DEVANAGARI LETTER A
u= ꣫ COMBINING DEVANAGARI LETTER U
k= ꣬ COMBINING DEVANAGARI LETTER KA
n= ꣭ COMBINING DEVANAGARI LETTER NA
p= ꣮ COMBINING DEVANAGARI LETTER PA
r= ꣯ COMBINING DEVANAGARI LETTER RA
s= ꣱ COMBINING DEVANAGARI SIGN AVAGRAHA
v= ꣰ COMBINING DEVANAGARI LETTER VI

Explicit Virama (Halant)

To type a Zero Width Non Joiner, _ can be typed between the consonants, for example: k _ S will generate क्‌ष.
