Onu! (Welcome!)

This is the Chwerty layout for writing Ibọnọ language. In this keyboard, accents are typed after the base letters.

So, to type "usem Ibọnọ" with this keyboard layout, simply tap the given letters and tap the appropriate accents to pace them on the letters.

Keyboard Layout

The keyboard uses CHWERTY layout.
