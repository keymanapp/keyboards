Quinault keyboard for Keyman

This keyboard layout is designed for Quinault. It includes an on screen keyboard.

Fonts

The keyboard will work with Times New Roman or many other Unicode encoded font. SIL International has produced several Unicode fonts that would also be suitable. These fonts are:

    Charis SIL
    Doulos SIL
    Andika
    Gentium Plus

Keyboard Layout

Default

Shift
