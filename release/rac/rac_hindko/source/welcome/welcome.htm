The Rachitrali-Hindko Keyman Keyboard Package

Rehmat Aziz Chitrali, 2018-12-05

Introduction

This keyboard is designed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral, Khyber Pakhtunkhwa of Pakistan for the Hindko language spoken in the Provinces of Khyber Pakhtunkhwa and Punjab, Pakistan. It is based on phonetic keyboard for Hindko Language developed by Pakistani linguist, author, researcher and Software Developer Mr.Rehmat Aziz Chitrali.

Font

The Rachitrali-Hindko Keyman keyboard is designed to work with any extended Arabic script font.

Keyboard

This keyboard layout is designed for the Hindko language. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.

ہندکو زبان

ہندکو زبان پاکستان کے صوبہ خیبر پختونخوا اور صوبہ پنجاب میں بولی جاتی ہے اس زبان میں جہاں رحمت عزیز چترالی کا ہندکو قاعدہ، ہندکو حروف تہجی، ہندکو خوشخطی، ہندکو سافٹ ویر، ہندکو اردو ذخیرہء الفاظ اور ہندکو ٹیکسٹ ایڈیٹر منظر عام پہ آ چکے ہیں وہاں ہندکو زبان بولنے والے نوجوانوں نے بھی اپنی مادری زبان ہندکو میں کتابیں لکھنے کا عزم ظاہر کیا ہے۔ کھوار اکیڈمی کے قیام کے بعد رحمت عزیز چترالی نے ہندکو زبان کے ساتھ ساتھ دیگر پاکستانی زبانوں کو کمپیوٹر پر تحریر کے قابل بنانے کے لیے کیبورڈ بنانے کا سلسلہ شروع کیا اور پاکستان کی چالیس زبانوں کو ایک ہی کیبورڈ کے زریعے تحریر کے قابل بناکر عالمی ریکارڈ قائم کرکے پاکستان کا نام عالمی سطح پر روشن کیا اور پاکستان میں بولی جانے والی زبان ہندکو کے لیے کیبورڈ بھی ایجاد کرنے کا سہرہ آپ ہی کے سر جاتا ہے اور آپ کے ایجاد کردہ کلیدی تختی کی مدد سے ہندکو زبان میں تحریر کا سلسلہ جاری ہے اور رحمت عزیز چترالی کا یہ کام ہندکو زبان کی تاریخ میں ایک سنگ میل کی حیثیت رکھتا ہے، پاکستانی زبانوں کے لیے بے لوث خدمات انجام دینے پر ہم حکومت پاکستان سے رحمت عزیز چترالی کے لیے ''ایوارڈ برائے حسن کارکردگی'' کی سفارش کرتے ہیں امید ہے کہ حکومت پاکستان رحمت عزیز چترالی اور ان جیسے دیگر پاکستانی ماہرین لسانیات اور ادیبوں کے لیے ایوارڈو اعزازات کا اعلان کرے گی، رحمت عزیز چترالی نے پہلی بار کھوار زبان اور ہندکو زبان کو گوگل کیبورڈ اور جی بورڈ میں شامل کرکے ان معدومیت کا شکار زبانوں کو بچانے کے لیے اہم کردار ادا کیا جو کہ زبانوں کی تاریخ میں سنہرے حروف سے لکھا جائے گا
