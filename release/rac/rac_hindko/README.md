Description
-----------

This keyboard is designed for the Hindko language spoken Pakistan, in its standard Perso-Arabic script, as used in Khyber Pakhtunkhwa Pakistan. It is based on phonetic keyboard for Hindko Language developed by Pakistani linguist, author, researcher and Software Developer Mr.Rehmat Aziz Chitrali.
