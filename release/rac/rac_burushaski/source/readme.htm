RAChitrali-Burushaski Keyman Keyboard

Rehmat Aziz Chitrali, 2017-10-07

Introduction

This keyboard was developed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral Khyber Pakhtunkhwa, Pakistan for keyboarding the Burushaski language in the Arabic Script. You can use this keyboard with any extended Arabic script Unicode font.
