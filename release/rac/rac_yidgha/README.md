Description
-----------

This keyboard is designed for the Yidgha language spoken in Chitral, Khyber Pakhtunkhwa, Pakistan, in its standard Perso-Arabic script, as used in Pakistan. It is based on phonetic keyboard for Yidgha Language developed by Pakistani linguist, author, researcher and Software Developer Mr.Rehmat Aziz Chitrali.
