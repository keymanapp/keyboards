Rachitrali-Hazaragi Keyman Keyboard

Rehmat Aziz Chitrali

Introduction

This keyboard was developed by Rehmat Aziz Chitrali for keyboarding the Hazaragi language in the Arabic Script. The Rachitrali-Hazaragi keyboard is designed to work with the Scheherazade font. However, you can use this keyboard with any extended Arabic script Unicode font.
