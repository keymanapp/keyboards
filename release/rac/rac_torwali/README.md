Description
-----------

This keyboard is designed for the Torwali language spoken in Bahrain and Chail of Swat District of Khyber Pakhtunkhwa Province of Pakistan, in its standard Perso-Arabic script, as used in Pakistan. It is based on phonetic keyboard for Torwali Language developed by Pakistani linguist, author, researcher, Inventor,lexicographer, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
