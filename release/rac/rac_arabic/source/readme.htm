RAChitrali-Arabic Keyman Keyboard

Introduction

This keyboard was developed by Rehmat Aziz Chitrali linguist and Researcher at Khowar Academy Chitral Khyber Pakhtunkhwa Pakistan for keyboarding the Arabic language in the Arabic Script. The Rachitrali-Arabic keyboard is designed to work with the Scheherazade New font. However, you can use this keyboard with any extended Arabic script Unicode font.
