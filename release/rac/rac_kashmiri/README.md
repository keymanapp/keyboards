Description
-----------

This keyboard is designed for the Kashmiri language spoken in the Kashmir, India and Pakistan, in its standard Perso-Arabic script, as used in Kashmir, India and Pakistan. It is based on phonetic keyboard for Kashmiri Language developed by Pakistani linguist, author, researcher, Inventor, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
