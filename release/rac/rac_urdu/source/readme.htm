Rachitrali-Urdu Keyman Keyboard

Rehmat Aziz Chitrali, 2018-06-25

Introduction

This keyboard was developed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral Pakistan for keyboarding the Urdu language in the Arabic Script. The Urdu keyboard is designed to work with the Scheherazade New font. However, you can use this keyboard with any extended Arabic script Unicode font.
