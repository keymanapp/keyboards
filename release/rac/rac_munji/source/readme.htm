RAChitrali-Munji Keyboard

Rehmat Aziz Chitrali, 2020-04-25

Introduction

This keyboard was developed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral Khyber Pakhtunkhwa, Pakistan for keyboarding the Munji language in the Arabic Script. You can use this keyboard with any extended Arabic script Unicode font.
