Description
-----------

This keyboard is designed for the Wadiyara language spoken in the Sindh Province  of Pakistan, in its standard Perso-Arabic script, as used in Pakistan. It is based on phonetic keyboard for Wadiyara Language developed by Pakistani linguist, author, researcher, Inventor, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
