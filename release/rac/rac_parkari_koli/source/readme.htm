RAChitrali Parkari Koli Keyman Keyboard

Rehmat Aziz Chitrali, 2018-06-03

Introduction

This keyboard was developed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral Khyber Pakhtunkhwa, Pakistan for keyboarding the Parkari Koli language in the Arabic Script. You can use this keyboard with any extended Arabic script Unicode font.
