Rachitrali-Dogri Keyboard Data
==============================

Copyright (c) Rehmat Aziz Chitrali

Description
-----------

This keyboard is designed for the Dogri language spoken Pakistan, Kashmir and India, in its standard Perso-Arabic script, as used in Asia region. It is based on phonetic keyboard for Dogri Language developed by Pakistani linguist, author, researcher and Software Developer Mr.Rehmat Aziz Chitrali.
