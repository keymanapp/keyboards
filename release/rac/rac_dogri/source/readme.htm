Rachitrali-Dogri Keyman Keyboard

Rehmat Aziz Chitrali, 2018-11-10

Introduction

This keyboard was developed by Rehmat Aziz Aziz Chitrali for keyboarding the Dogri language in the Arabic Script. The Rachitrali-Dogri keyboard is designed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral, Khyber Pakhtunkhwa Pakistan to work with the Scheherazade New font. However, you can use this keyboard with any extended Arabic script Unicode font.
