Description
-----------

This keyboard is designed for the Indus-Kohistani language spoken in Indus-Kohistan of Khyber Pakhtunkhwa, Pakistan, in its standard Perso-Arabic script, as used in Pakistan. It is based on phonetic keyboard for Indus-Kohistani Language developed by Pakistani linguist, author, researcher, Inventor, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
