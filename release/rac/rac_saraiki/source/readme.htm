Rachitrali-Saraiki Keyman Keyboard

Rehmat Aziz Chitrali 2018-06-14

Introduction

This keyboard was developed by Rehmat Aziz Chitrali for keyboarding the Saraiki language in the Arabic Script. The Rachitrali-Saraiki keyboard is designed to work with the Scheherazade New font. However, you can use this keyboard with any extended Arabic script Unicode font.
