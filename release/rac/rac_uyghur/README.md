Rachitrali-Uyghur Keyboard Data
===============================

Copyright (c) Rehmat Aziz Chitrali

Description
-----------

This keyboard is designed for the Uyghur language spoken China, in its standard Perso-Arabic script, as used in  Xinjiang China. It is based on phonetic keyboard for Uyghur Language developed by Pakistani linguist, author, researcher and Software Developer Mr.Rehmat Aziz Chitrali.
