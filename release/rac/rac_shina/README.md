Description
-----------

This keyboard is designed for the Shina language spoken in Gilgit-Baltistan Pakistan, in its standard Perso-Arabic script, as used in Pakistan. It is based on phonetic keyboard for Shina Language developed by Pakistani linguist, author, researcher, Inventor, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
