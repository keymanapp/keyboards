RAChitrali-Shina Keyman Keyboard

Introduction

This keyboard was developed by Rehmat Aziz Chitrali Linguist and Researcher at Khowar Academy Chitral Khyber Pakhtunkhwa, Pakistan for keyboarding the Shina language in the Arabic Script. You can use this keyboard with any extended Arabic script Unicode font.
