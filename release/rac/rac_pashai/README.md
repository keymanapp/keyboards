Description
-----------

This keyboard is designed for the Pashai language spoken in Afganistan, in its standard Perso-Arabic script, as used in Afghanistan. It is based on phonetic keyboard for Pashai Language developed by Pakistani linguist, author, researcher, Inventor, Scientist and Software Developer Mr.Rehmat Aziz Chitrali.
