Start Using Warang Citi Keyboard

Start typing Warang Citi phonetically according to English alphabet.

Rules for Desktop

Type DH for Warang Citi letter DA:
Type TH for Warang Citi letter OT
Type NG for Warang Citi letter ONG
Type NJ for Warang Citi letter BINJ
Type NN for Warang Citi letter ENN
Type RR for Warang Citi letter HORR
Type V for Warang Citi letter ANG
Type X for Warang Citi letter SSU
Type @ for Warang Citi letter OWANG
Type CTRL+DIGITS for Arabic Num

Rules for Mobile Devices

Type D for Warang Citi letter DA:
Type Ḑ for Warang Citi letter ODD
Type T for Warang Citi letter OT
Type Ț for Warang Citi letter TTE
Type NG for Warang Citi letter ONG
Type NJ for Warang Citi letter BINJ
Type NN for Warang Citi letter ENN
Type RR for Warang Citi letter HORR
Type V for Warang Citi letter ANG
Type X for Warang Citi letter SSU
