Fonts

There is a problem with many fonts not stacking diacritics correctly.

Charis SIL displays all the stacked diacritics correctly except the U+0300 and U+0301 sequence.

In this version of the keyboard we've changed ; to U+02CA to help with this problem. We are working on a better solution.
