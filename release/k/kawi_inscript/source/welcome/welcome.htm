Start Using Kawi Inscript

This is a keyboard for the Kawi script using an Inscript-based layout. Feedback is welcome.

For ease in typing, the RALT and RALT+SHIFT layers produce conjunct forms (conjoiner plus consonant). The conjoiner can also be typed separately on the "\" key.

This keyboard is distributed with the Noto Sans Kawi font. Although the keyboard supports a nukta (on the "]" key) the font does not have a nukta character. It will display as a square box.
