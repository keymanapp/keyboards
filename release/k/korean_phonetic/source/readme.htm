Korean Phonetic

The Korean Phonetic keyboard creates Korean hangeul from jamo each time you press Spacebar.

The Korean Phonetic keyboard is built to type Korean using transliteration system when possible. This layout is designed to make Latin-Korean transliteration 100% unambiguous.

This mnemonic keyboard is based on the pronunciation of Korean. For an alternative accurate transliteration system, based on Korean pronunciation, use the Korean KORDA keyboards.
