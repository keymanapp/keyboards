Start Using Karakalpak Cyrillic

This keyboard is designed for Cyrillic script of Karakalpak Language.

Keyboard Layout

Desktop Layout

This keyboard utilizes a deadkey on the backslash key (\). To get the desired character, press the deadkey and the character to the next shown in Key combination row in the below table.

For example, if you want to get letter 'ә', you need to first type '\', then 'а'.

To access the "deadkey" itself (\), type a space after it, or double tap it.

  Key combination     Desired character  
\ + а ә
\ + г ғ
\ + к қ
\ + н ң
\ + о ө
\ + у ў
\ + и ү
\ + х ҳ
\ + [space] \
