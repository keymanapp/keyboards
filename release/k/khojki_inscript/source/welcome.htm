Start Using khojki_inscript

This keyboard is designed for the Sindhi language, using the Khojki script with an Inscript layout.

Keyboard Layout

Desktop

Default (unshifted)

Shift

The numeric row on the shift layer is used to type the signs of the Common Indic Number Forms.

Phone

Default (unshifted)

Shift

Numeric

The second row on the numeric layer is used to type the signs of the Common Indic Number Forms.
