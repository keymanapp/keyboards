Start Using kharoshthi_inscript

The Kharoshthi (or Kharosthi) script was invented during the 3rd century BC by Gandhari people. It was used to write Sanskrit, Gandhari, Pali and other languages used along the Silk Road.
This keyboard is designed for the Sanskrit language, using the Kharoshthi script with an Inscript layout.
