Luri Paprik Keyboard

A Keyman keyboard for typing Northern Luri (lrc-Arab) on desktop and mobile platforms.

Desktop Layers

  • Default: Luri characters (ؽ, ۋ, گ, ڤ, ۉ, ۏ, ۊ, ؤ, لٛ, اٛ, etc.)
  • Shift: Arabic diacritics (Fatha, Damma, Kasra, Shadda, Sukun), special letters (ڳ, ڒ, ڌ, لٛ, إ), punctuation (؛ ، ٪ — « »), and Persian letters
  • Right Alt: Math symbols (> < ≤ ≥), brackets (⟨ ⟩ ﴿ ﴾ ‹ ›), currency (£ ¥ ¢), Arabic decimal separators (٫ ٬), interrobang (‽)
  • Right Alt + Shift: Not currently assigned

Touch

Long-press on base keys to access Luri variants. Use the symbol toggle key for currency, brackets, and punctuation.

ZWNJ

Right Alt + Space produces Zero-Width Non-Joiner (ZWNJ).
