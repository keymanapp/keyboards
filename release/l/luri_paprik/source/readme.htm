Luri Paprik Keyboard

A Keyman keyboard for typing Northern Luri (lrc-Arab) on desktop and mobile platforms.

Desktop Layers

  • Default: Luri characters (ؽ, ۋ, گ, ڤ, ۉ, ۏ, ۊ, ؤ, لٛ, اٛ, کٛ, etc.)
  • Shift: Persian ISIRI 9147 layout with Extended Arabic-Indic digits on number row
  • Right Alt: Punctuation, math symbols, currency (€, £, ¥, ¢, ﷼), ornate parentheses
  • Right Alt + Shift: Persian letters, Arabic diacritics (Fatha, Damma, Kasra, Shadda, Sukun), Tatweel

Touch

Long-press on base keys to access Luri variants. Use the symbol toggle key for currency, brackets, and punctuation.

ZWNJ

Right Alt + Space produces Zero-Width Non-Joiner (ZWNJ).
