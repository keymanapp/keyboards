Start Using Lamkaang

To add a Lamkaang acute accent or dot to any letter, after typing the letter, press the backtick (`) key, normally at the top left corner of the keyboard.

e.g. Typing a` produces á. Typing t` produces ṭ.

To add the ring accent mark, press the backtick key twice.

e.g. Typing u`` produces ů.
