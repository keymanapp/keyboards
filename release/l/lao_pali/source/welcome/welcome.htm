Lao Pali Keyboard

This package includes the Lao Pali keyboard and the font SengBuhan.

The keyboard uses the official Lao standard layout, adapted to support entry of the Lao Pali and Lao Sanskrit characters included in Unicode 13. The keyboard includes some overtyping constraints but minimal re-ordering and no automatic break insertion. Break characters can, however, be inserted manually with |.

The SengBuhan font also includes all characters output by the keyboard.
