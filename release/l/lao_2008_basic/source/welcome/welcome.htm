The Lao 2008 Keyboards and Font

Thank you for downloading and installing the Lao 2008 keyboard/font package for Tavultesoft Keyman.

The package includes the Lao Unicode (OpenType) font Saysettha OT and the following two Lao keyboards:

KeyboardIconFeatures
Lao 2008 Basic Prevents accidentally overtyping vowel or tone marks.
No re-ordering or automatic break insertion.
Word-wrap by manual entry of break characters at syllable or word boundaries.
Usable with all Unicode-compliant Windows applications.
Lao 2008 Rapid Prevents accidentally overtyping vowel or tone marks.
Automatic re-ordering of vowel and tone marks.
Word-wrap by automatic insertion of hidden break characters at syllable boundaries.
Usable with most Unicode-compliant Windows applications.

Both keyboards output Lao Unicode characters according to the the official Lao standard layout, and may be used with any Lao Unicode font.
Hidden break characters (ZWSP, U+200B) can be inserted manually (with either keyboard) by using the | key.

Using Lao Unicode fonts

In Microsoft Word and some other Microsoft applications, when selecting the Saysettha OT font (or other Lao Unicode fonts), it is usually necessary to set both the Latin text font and the Complex script font to Saysettha OT. A simple way to do this in Word is to create a paragraph (or character) style, and set the font for "all scripts" to be Saysettha OT. If the new style is added to your Normal template, it can be applied whenever entering Lao text.
