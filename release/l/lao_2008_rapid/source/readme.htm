Lao 2008 rapid Keyboard

This package includes the Lao 2008 Rapid keyboard and the Lao Unicode (OpenType) font Saysettha OT.

The keyboard uses the official Lao standard layout and output Lao according to the Unicode standard. The Lao 2008 Rapid keyboard also automatically inserts (hidden) break characters at syllable boundaries to manage line breaking in Lao text entered without spaces. Break characters can also be inserted manually with either keyboard by using the | key.

See the keyboard layout information or the On Screen Keyboard for more information.
