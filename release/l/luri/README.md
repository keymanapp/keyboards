# Luri Keyboard

A Keyman keyboard for Southern Luri (ISO 639-3: `luz`, BCP 47: `luz-Latn`).

Supports desktop (Windows, macOS, Linux) and mobile (Android, iOS) platforms.

## Layout

- **Desktop**: Standard QWERTY + RALT combinations for Luri-specific characters.
- **Touch**: Long-press menus on relevant keys for accented variants.

## Building

```sh
kmc build luri.kpj
```

Output is written to the `build/` directory.
