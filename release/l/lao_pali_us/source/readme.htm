Lao Pali US

The Lao Pali (US) keyboard package includes the Lao Pali (US) keyboard and the font SengBuhan and follows the "Lao Original (Lao Pali)" key layout for easier typing for users familiar with the standard US English keyboard. The keyboard supports entry of all Lao Pali and Lao Sanskrit characters included in Unicode 13.
