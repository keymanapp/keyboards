ግዕዝ (Ge'ez) Keyboard
====================

Copyright (C) 2009-2017 Ge'ez Frontier Foundation

Version 1.1

This is a Ge'ez (ግዕዝ, ISO-639-2 gez) language mnemonic input method.  It requires a font
supporting Ethiopic script under the Unicode 3.0 standard.
