ግዕዝ (Geʾez) Keyboard
====================

Copyright © 2009-2023 Geʾez Frontier Foundation

Version 2.0.1

This is a Geʾez (ግዕዝ, ISO-639-2 gez) language mnemonic input method.  It requires a font
supporting Ethiopic script under the Unicode 4.1 standard.
