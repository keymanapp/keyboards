OR Vietnamese (VIQTR) Unicode Keyboard Layout

This keyboard layout was designed for typing Vietnamese with Keyman. It is a modified version of a keyboard layout originally developed for Vietnamese Ethnic Minority languages that use the Latin script. It uses Vietnamese Quoted-Readable 7-bit transcription system (VIQR) as the input mechanism.

How to type in Vietnamese

Diacritical Marks in VIQR
Diacritical markTyped characterExamples
trăng (breve)(a( → ă
mũ (circumflex)^a^ → â
móc (horn)+o+ → ơ
huyền (grave)`a` → à
sắc (acute)'a' → á
hỏi (hook)?a? → ả
ngã (tilde)~a~ → ã
nặng (dot below).a. → ạ

VIQR uses DD for the Vietnamese letter Đ, and dd for the Vietnamese letter đ. Alternatively you could type D- or d-.

To type certain punctuation marks (namely, the period, question mark, apostrophe, forward slash, opening parenthesis, or hyphen) directly after Vietnamese words ending in a vowel or the letter d, a backslash (\) must be typed directly before the punctuation mark, so that it will not be interpreted as a diacritical mark.
