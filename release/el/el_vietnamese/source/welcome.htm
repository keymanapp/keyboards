OR Vietnamese Unicode Keyboard Layout

This keyboard layout was designed for typing Vietnamese with Keyman. It is a modified version of a keyboard layout originally developed for Vietnamese Ethnic Minority languages that use the Latin script. The layout is derived from the Vietnamese layout used by Microsoft on Windows 2000/XP/2003. Microsoft's keyboard layout uses combining diacritics for each of the five tone markers.

This version of the layout generates single precomposed letters rather than using a vowel and a combining diacritic.

Keyboard layouts:

Keyboard layout designed for Vietnamese. Uses the Microsoft's Vietnamese keyboard layout, but produces precomposed Unicode characters.
