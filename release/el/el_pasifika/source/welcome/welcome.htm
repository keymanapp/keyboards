Typing Igbo with tones

To type tones, type the vowel followed by the diacritic key:

  1. Acute: type the vowel then the equals key, e.g. (a= yields á)
  2. Grave: type the vowel then the plus key, e.g. (a+ yields à)
  3. Macron: type the vowel then the righ bracket key, e.g. (a] yields ā)

Fonts

The Doulos SIL font is installed with the Igbo keyboard layout. Additional fonts include:

  • Charis SIL;
  • Gentium (if tone marking is not required).

For information on installing fonts in Windows XP, please read the MyLanguage font installation guide.
