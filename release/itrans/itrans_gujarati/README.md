Gujarati Phonetic (ITRANS) 
=================

© sanskritdocuments.org

Description
-----------

A Phonetic keyboard for Gujarati using ITRANS transliteration scheme. 
Examples: Type `saMskRta` for સંસ્કૃત, `hindI` for હિન્દી, `gujarAtI` for ગુજરાતી.
