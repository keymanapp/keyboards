Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard

This is a phonetic Devanagari keyboard layout with support for Vedic Sanskrit (Devanagari Extended and Vedic Extensions Unicode ranges). You need to use a Unicode Devanagari font which has support for these ranges, e.g. Siddhanta font by Mihail Bayaryn which is included in this package.

Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard is based on the phonetic ITRANS transliteration scheme by Avinash Chopde, with some minor modifications.

For detailed information about Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard, visit https://sanskritdocuments.org/learning_tools/sanskritvedic.html
