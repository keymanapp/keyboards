Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard

This is a phonetic Devanagari keyboard layout with support for Vedic Sanskrit (Devanagari Extended and Vedic Extensions Unicode ranges). You need to use a Unicode Devanagari font which has support for these ranges, e.g. Siddhanta font by Mihail Bayaryn.

Vedic Sanskrit Devanagari Phonetic (ITRANS) Keyboard is based on the phonetic ITRANS transliteration scheme by Avinash Chopde, with some minor modifications.

Thanks to P Narayanaswami, Ken P and Lorna Evans for testing the keyboard and providing feedback .
