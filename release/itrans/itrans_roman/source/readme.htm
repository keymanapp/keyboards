Indic Romanization from ITRANS Keyboard Version 1.0.0

This is a phonetic keyboard for Indic Romanization which is based on ITRANS transliteration scheme with some modifications for input. 7bit ISO scheme has also been referred to for modifying the input scheme. Accented letters used by various Indic Romanization schemes such as ISO, IAST, NLK are supported for output. It also supports typing of IPA vowels for use in pronunciation keys.

The modified input scheme uses only lowercase letters. This allows the same letters to be typed with SHIFT to get Unicode Romanized output in UPPERCASE letters, which is useful for typing Proper names, Headings, etc.
