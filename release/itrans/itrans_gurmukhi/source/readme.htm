Gurmukhi Phonetic (ITRANS) Keyboard

This is a phonetic keyboard layout for Gurmukhi. It is based on the ITRANS transliteration scheme by Avinash Chopde, with some minor modifications. This can be used for writing Punjabi and Sindhi languages. Examples: Type `hindI` for ਹਿਨ੍ਦੀ, `hiMdI` for ਹਿੰਦੀ,`gujarAtI` for ਗੁਜਰਾਤੀ, `paMjAbI` for ਪੰਜਾਬੀ.
