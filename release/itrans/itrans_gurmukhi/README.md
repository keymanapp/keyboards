Panjabi/Gurmukhi Phonetic (ITRANS)
=================

©2019 sanskritdocuments.org

Description
-----------

A Phonetic keyboard for Gurmukhi using ITRANS transliteration scheme. 
Examples: Type `hindI` for ਹਿਨ੍ਦੀ, `gujarAtI` for ਗੁਜਰਾਤੀ, `panjabI` for ਪੰਜਾਬੀ.


Supported Platforms
-------------------
* Windows
* macOS
* Linux
* Web
