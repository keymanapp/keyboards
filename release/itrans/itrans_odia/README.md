Odia/Oriya Phonetic (ITRANS) 
=================

©2019 sanskritdocuments.org

Description
-----------

A Phonetic keyboard for Odia/Oriya using ITRANS transliteration scheme. 
Examples: Type `saMskRta` for ସଂସ୍କୃତ, `hindI` for ହିନ୍ଦୀ, `gujarAtI` for ଗୁଜରାତୀ.
