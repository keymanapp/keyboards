Nias keyboard
==============

©2021 Sirus Laia

Version 1.0

Description
-----------

Nias keyboard with ö and ŵ characters. Press [ produces ö and ] produces ŵ. To get [ press Alt-Right+[ and to get ] press Alt-Right+]. Similar for getting { and } (Alt-Right+Shift).
