Welcome to Nias keyboard/Selamat menggunakan keyboard Nias

On PC: click [ to get ö and ] to get ŵ. Hold together ALT-RIGHT+[ and ALT+RIGHT+] to get [ and ]. Add SHIFT to it to get the { and }. See the picture below.

Di laptop/desktop: klik tombol [ untuk menghasilkan ö dan tombol ] untuk menghasilkan ŵ. Tekan bersama-sama ALT-KANAN+[ atau ALT-KANAN+] untuk mengetik [ dan ] sementara SHIFT+ALT-KANAN+[ dan SHIFT+ALT-KANAN+] untuk mengetik tanda { dan }. Lihat gambar papan ketik di bawah.

Source code/Kode sumber: https://github.com/sslaia/keyboards (lihat folder release/n/nias)
