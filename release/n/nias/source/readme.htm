Nias keyboard

On PC: press [ to get ö and ] to get ŵ. Press Alt-Right+[ and Alt+Right+] to get [ and ]. Add Shift to it to get the { and }. See the picture below.

Di laptop/desktop: tombol [ menghasilkan ö dan ] menghasilkan ŵ. Tekan Alt-Kanan+[ atau Alt-Kanan+] untuk mengetik [ dan ] sementara Shift+Alt-Kanan+[ dan Shift+Alt-Kanan+] untuk mengetik tanda { dan }. Lihat gambar papan ketik di bawah.

Source code/Kode sumber: https://github.com/sslaia/keyboards/release/n/nias
