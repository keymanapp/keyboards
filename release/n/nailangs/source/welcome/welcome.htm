Introduction

This keyboard layout is designed for languages of Nigeria. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.

The Nailangs keyboard is designed to support and enable typing at least 12 Nigerian languages correctly using correct letters and tone marks which help in imparting instant meaning to the reader as these languages are tonal languages:
