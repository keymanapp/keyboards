Start using the Northern Sierra Miwok Keyboard

This keyboard is for typing Northern Sierra Miwok using the alphabet adopted by the Shingle Springs Band of Miwok Indians (SSBMI) for use in Northern Sierra Miwok language materials.

 

Keyboard Layout

The Northern Sierra Miwok alphabet uses a number of special symbols that are not available on a standard keyboard.

With the Northern Sierra Miwok Keyboard enabled, you can type these symbols on a physical keyboard device by holding RIGHT ALT/OPTION or RIGHT ALT/OPTION+SHIFT and pressing the corresponding key.
