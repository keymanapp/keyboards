Start using the Nisenan Keyboard

This keyboard is for typing Nisenan using the alphabet adopted by the Shingle Springs Band of Miwok Indians (SSBMI) for use in Nisenan language materials.

 

Keyboard Layout

The Nisenan alphabet uses a number of special symbols that are not available on a standard keyboard.

With the Nisenan Keyboard enabled, you can type these symbols on a physical keyboard device by holding CTRL/CMD or CTRL/CMD+SHIFT and pressing the corresponding key.
