Numanggang keyboard
==============

© SIL Global

Requires Keyman 17 or greater

Description
-----------

Keyboard for typing the Numanggang language of Papua New Guinea (ISO 639-3: nop), written in Latin script. Created by David Hynum.

Usage
-----

 * `;` → `ŋ` (eng, U+014B). Press `;` again for a literal `;`; a third press gives `ŋŋ`.
 * `|` → `Ŋ` (eng, U+014A). Press `|` again for a literal `|`; a third press gives `ŊŊ`.
