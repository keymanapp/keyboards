Type the ; key to get the "ŋ" character. Type the ; key again to get the ";" character. Type the ; key a third time to get "ŋŋ".

Type the | key to get the "Ŋ" character. Type the | key again to get the "Ŋ" character. Type the | key a third time to get "ŊŊ".
