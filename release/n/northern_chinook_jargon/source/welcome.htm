Start Using Northern Chinook Jargon

Keyman keyboard for Northern / BC Chinook Jargon using Dr David Douglas Robertson's latin orthography. For more information see his site https://chinookjargon.com

Keyboard Layout

This is a standard asci keyboard with the following additions:

  • SHIFT + right ALT + h -> 'H̱'
  • right ALT + h -> 'ẖ'
  • SHIFT + right ALT + k -> 'Ḵ'
  • right ALT + k -> 'ḵ'
  • SHIFT + right ALT + w -> 'W̱H̱'
  • right ALT + w -> 'w̱ẖ'
  • SHIFT + right ALT + x -> 'X̱'
  • right ALT + x -> 'x̱'
  • right ALT + \ -> 'ʼ' (Unicode U+02BC "MODIFIER LETTER APOSTROPHE")

Mobile Keyboard Layout

The mobile layout uses long press to access letters as follows

  • long press H -> 'H̱'
  • long press h -> 'ẖ'
  • long press K -> 'Ḵ'
  • long press k -> 'ḵ'
  • long press W -> 'W̱H̱'
  • long press w -> 'w̱ẖ'
  • long press X -> 'X̱'
  • long press x -> 'x̱'

Punctuation appears as a list when long pressing "." The apostrophe modifier for glottals 'ʼ' (Unicode U+02BC "MODIFIER LETTER APOSTROPHE") is on the list between the single and double quote.

This package does not include any fonts. It includes one keyboard "Northern Chinook Jargon".
