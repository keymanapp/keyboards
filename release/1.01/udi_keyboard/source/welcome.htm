Udi Keyboard Help

This keyboard is designed for the Udi language of Azerbaijan, Georgia and Russia.  It uses standard Windows fonts.

This keyboard makes use of modifier keys. Typing the modifier key before the base character will output the new or combined character. For example:

[c (Open Bracket and c) will produce č (available on cC eE əƏ gG iİ jJ oO sS uU).

]o (Close Bracket and o) will produce ö ( available on oO uU)

]s (Close Bracket and s) will produce ś

`c (Back quote and c) will produce ç (available on sS).

‘c (Quote and c) will produce cʹ (available on cC kK qQ pP sS tT).

< and < will produce «

> and > will produce »

=c (equal sign and c) will produce ç̌ʹ (available on cC)

If square boxes are displayed instead of characters when using this keyboard (and in the keyboard layouts below), please read our troubleshooting guide.
