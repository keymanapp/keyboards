Este es un teclado móvil para el triqui de Itunyoso.

Es similar a un teclado móvil estándar, con algunas diferencias en las teclas.

Algunas letras no se usan en triqui, por lo que se han reemplazado por teclas de dos y tres letras que representan sonidos individuales en triqui.

Puede escribir tonos usando las teclas de acento a la derecha del teclado, escribiéndolas antes de una vocal.

También puede escribir tonos deslizando hacia arriba o hacia abajo sobre una vocal, manteniendo presionada una vocal o usando la función de texto predictivo.

Puede instalar la función de texto predictivo por separado, después de instalar el teclado.

This is a mobile keyboard for Itunyoso Triqui.

It is similar to a standard mobile keyboard, with some key differences.

Some letters are not used in Triqui, so they have been replaced with two- and three-letter keys that represent individual sounds in Triqui.

You can type tones using the accent keys to the right of the keyboard, typing them before a vowel.

You can also type tones by swiping up or down on a vowel, long-pressing a vowel, or using the predictive text feature.

You can install the predictive text feature separately, after installing the keyboard.
