Tai Nua  keyboard
=====================

Copyright (C) 2016, 2018 Sai Ai Wong

Version 1.0.2

An Unicode keyboard for the Tai Nua language (tdd). Only the Tai Nua unicode range is supported.
Please switch to a different keyboard to type English.
