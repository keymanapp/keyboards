# Tawrã-Kaman Keyboard

This keyboard is designed for the Tawrã (Taraon) Mishmi and Kaman languages.

## Keyboard Layout
This keyboard follows a phonetic layout suitable for typing Tawrã and Kaman characters for mobile devices.

## Fonts
The keyboard package includes the necessary fonts for proper rendering of the characters.
