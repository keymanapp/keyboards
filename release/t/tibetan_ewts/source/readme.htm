Tibetan EWTS Readme

Overview

Tibetan EWTS is based on the Extended Wylie Transliteration Scheme (EWTS), an approach that avoids most Shift, Alt, and Alt+Shift combinations.

Basic Function

Tibetan characters are created from Latin letters each time you type a vowel or a Space (tsheg). Special rules apply for long-stacks.

This keyboard also features:

  • Menus for extended Tibetan symbols

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Tibetan script. The Tibetan Machine Uni font supports Tibetan script and is installed with the keyboard. To find other fonts on your computer which support this keyboard, use the Keyman Font Helper.
