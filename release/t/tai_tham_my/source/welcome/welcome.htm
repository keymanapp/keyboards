• ᩅᩥᨳᩦᨩᩱᩢ ᨤᩦᨷ᩠ᩋᨯᨴᩱ᩠ᨿᨵᨾ᩠ᨾ᩼ᨾᩮᩨ᩠ᨦᨿ᩠ᩋᨦ

    ᨤᩦᨷ᩠ᩋᨯᨴᩱ᩠ᨿᨵᨾ᩠ᨾ᩼ᨾᩮᩨ᩠ᨦᨿ᩠ᩋᨦᨶᩦᩢ ᩌᩮᨯᨾᩣᨻᩮᩨ᩠᩵ᩋᨩᩱᩢᨽᩥ᩠ᨾ ᨣᩣᩴᨴᩱ᩠ᨿᩃᩨᩢ ᨯ᩠ᩅ᩠ᨿᩢ ᨲᩫ᩠ᩅᨴᩱ᩠ᨿᨵᨾ᩠ᨾ᩼᪫ ᨤᩦᨷ᩠ᩋᨯᨶᩦᩢᨩᩱᩢᨯᩱᨴᩨ᩠ᨦᨴᩦ᩵ᨶᩱ ᨤ᩠ᩋᨾ ᩓ ᨶᩱᨾᩨᨳᩨ᪫

    ᨸᩯ᩠ᨶᩢᨽᩦ᩠ᨾ ᨤᩦᨷ᩠ᩋᨯ

    ᩅᩥᨳᩦᨩᩱᩢᨶᩱ ᨤ᩠ᩋᨾᨽᩥ᩠ᩅᨴᩮᩨᩢ᩠ᩋ

    ᨶᩱᨤ᩠ᩋᨾᨠᩣ᩠ᩁᨩᩱᩢᨾᩦ ᪃ ᩀ᩵ᩣ᩠ᨦ᪫ ᨠᩣ᩠ᩁᨽᩥ᩠ᨾᨵᨾ᩠ᨾ᩼ᨯᩣ᪪ ᨽᩥ᩠ᨾᨠᩢ᩠ᨷᨲᩫ᩠ᩅ Shift ᩓ ᨩᩱᩢᨽᩥ᩠ᨾᨠᩢ᩠ᨷᨲᩫ᩠ᩅ "Alt" ᨲᩦ᩵ᨾᩦᨻᩣ᩠ᨿᨠ᩠ᩅᩣ RightAlt

    1. ᨲᩫ᩠ᩅ A ᨸᩮ᩠ᨶ "ᨼ" ᪪ ᨲᩫ᩠ᩅ B ᨸᩮ᩠ᨶ "ᨠᩥ"
    2. ᩮ ᩯ ᩱ ᩰ ᩕ ᨶᩦᩢᩉᩨᩢᨲᩯ᩠ᨾᩢᩉᩖᩢᨦᨻ᩠ᨿᨬ᩠ᨩᨶ ᩉᩮ᩠ᨾᩨᩁ ᨠ + ᩮ ᨸᩮ᩠ᨶ ᨠᩮ
    3. ᨣᩢ᩠ᨶᨤᩱ᩵ᨲᩯ᩠ᨾᩢᨲᩫ᩠ᩅᨲᩱᩢᩉᩨᩢᨲᩯ᩠ᨾᩢᨲᩫ᩠ᩅᩉᩮ᩠ᨶᩨᩋᩋ᩠ᩋᩁᩓ᩠ᩅᩢᨧᩨ᩠᩵ᨦᨩᩱᩢᨲᩫ᩠ᩅᨪᩢ᩠ᩋᩁ " ᩠" ᩓ᩠ᩅᩢᨣᩢ ᨽᩥ᩠ᨾᨲᩫ᩠ᩅᨲᩱᩢ ᩉᩮ᩠ᨾᩨᩁ ᨴ + ᩱ + ᩠ + ᨿ ᨸᩮ᩠ᨶ ᨴᩱ᩠ᨿ ᪫

    ᩅᩥᨳᩦᨽᩥ᩠ᨾᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Shift“ (ᨤ᩠ᩋᨾ)

    ᨻ᩠ᨿᨬ᩠ᨩᨶᩋᩢ᩠ᨶᩉᩯ᩠ᨶᩅᩱᩢᨻᩣ᩠ᨿᨲᩱᩢᨶᩦᩢ ᨧᩢᨯᩱᩢᨩᩱᩢᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Shift” ᪫ ᩉᩮ᩠ᨾᩨᩁ Shift + A ᨸᩮ᩠ᨶ ᩀ

    ᩅᩥᨳᩦᨽᩥ᩠ᨾᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Right Alt” (ᨤ᩠ᩋᨾ)

    ᨻ᩠ᨿᨬ᩠ᨩᨶᩋᩢ᩠ᨶᩉᩯ᩠ᨶᩅᩱᩢᨻᩣ᩠ᨿᨲᩱᩢᨶᩦᩢ ᨧᩢᨯᩱᩢᨩᩱᩢᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Right Alt” ᪫ ᩉᩮ᩠ᨾᩨᩁ Right Alt + 1 ᨸᩮ᩠ᨶ ᪑

    ᩅᩥᨳᩦᨩᩱᩢᨶᩱ ᨾᩨᨳᩨ

    ᨶᩱᨾᩨᨳᩨᨠᩣ᩠ᩁᨩᩱᩢᨾᩦ ᪃ ᩀ᩵ᩣ᩠ᨦ᪫ ᨠᩣ᩠ᩁᨽᩥ᩠ᨾᨵᨾ᩠ᨾ᩼ᨯᩣ᪪ ᨽᩥ᩠ᨾᨠᩢ᩠ᨷᨲᩫ᩠ᩅ Shift ᩓ ᨩᩱᩢᨽᩥ᩠ᨾᨠᩢ᩠ᨷᨲᩫ᩠ᩅ "Alt” ᨲᩦ᩵ᨾᩦᨻᩣ᩠ᨿᨠ᩠ᩅᩣ RAlt᪫

    ᩅᩥᨳᩦᨽᩥ᩠ᨾ ᨵᨾ᩠ᨾ᩼ᨯᩣ (ᨾᩨᨳᩨ)

    ᨠᩣ᩠ᩁᨩᩱᩢᨸᩯ᩠ᨶᩢᨽᩥ᩠ᨾᨵᨾ᩠ᨾ᩼ᨯᩣ ᩉᩯ᩠ᨶᩅᩱᩢᨻᩣ᩠ᨿᨲᩱᩢᨶᩦᩢ᪫ ᨲᩣ᩠ᨾᨯ᩠᩶ᨦᨶᩱᨸᩯ᩠ᨶᩢᨽᩥ᩠ᨾᨵᨾ᩠ᨾ᩼ᨯᩣᨶᩦᩢ ᨣᩢ᩠ᨶᨽᩥ᩠ᨾ ᨻ᩠ᨿᨬ᩠ᨩᨶᨲᩫ᩠ᩅ ᨯᩱᨣᩢᨧᩢᨸᩮ᩠ᨶᨲᩫ᩠ᩅᨶ᩠ᨶᩢᩢ᪫ ᩉᩮ᩠ᨾᩨᩁ

    1. ᨲᩫ᩠ᩅ A ᨸᩮ᩠ᨶ "ᨼ" ᪪ ᨲᩫ᩠ᩅ B ᨸᩮ᩠ᨶ "ᨠᩥ"
    2. ᩮ ᩯ ᩱ ᩰ ᩕ ᨶᩦᩢᩉᩨᩢᨲᩯ᩠ᨾᩢᩉᩖᩢᨦᨻ᩠ᨿᨬ᩠ᨩᨶ ᩉᩮ᩠ᨾᩨᩁᨠ + ᩮ ᨸᩮ᩠ᨶ ᨠᩮ
    3. ᨣᩢ᩠ᨶᨤᩱ᩵ᨲᩯ᩠ᨾᩢᨲᩫ᩠ᩅᨲᩱᩢᩉᩨᩢᨲᩯ᩠ᨾᩢᨲᩫ᩠ᩅᩉᩮ᩠ᨶᩨᩋᩋ᩠ᩋᩁᩓ᩠ᩅᩢᨧᩨ᩠᩵ᨦᨩᩱᩢᨲᩫ᩠ᩅᨪ᩠ᩋᩢᩁ " ᩠" ᩓ᩠ᩅᩢᨣᩢ ᨽᩥ᩠ᨾᨲᩫ᩠ᩅᨲᩱᩢ ᩉᩮ᩠ᨾᩨᩁ ᨴ + ᩱ + ᩠ + ᨿ ᨸᩮ᩠ᨶ ᨴᩱ᩠ᨿ ᪫

    ᩅᩥᨳᩦᨽᩥ᩠ᨾᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Shift“ (ᨾᩨᨳᩨ)

    ᨻ᩠ᨿᨬ᩠ᨩᨶᩋᩢ᩠ᨶᩉᩯ᩠ᨶᩅᩱᩢᨻᩣ᩠ᨿᨲᩱᩢᨶᩦᩢ ᨧᩢᨯᩱᩢᨩᩱᩢᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Shift” ᪫ ᩉᩮ᩠ᨾᩨᩁ Shift + A ᨸᩮ᩠ᨶ ᩀ

    ᩅᩥᨳᩦᨽᩥ᩠ᨾᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Right Alt” (ᨾᩨᨳᩨ)

    ᨻ᩠ᨿᨬ᩠ᨩᨶᩋᩢ᩠ᨶᩉᩯ᩠ᨶᩅᩱᩢᨻᩣ᩠ᨿᨲᩱᩢᨶᩦᩢ ᨧᩢᨯᩱᩢᨩᩱᩢᨩ᩠ᩋᨾᨲᩫ᩠ᩅ "Right Alt” ᪫ ᩉᩮ᩠ᨾᩨᩁ RAlt + 1 ᨸᩮ᩠ᨶ ᪑

  • Start Using Tai Tham MY Keyboard

    The Tai Tham MY keyboard was developed for typing Tai Lue language in Tai Tham script. The keyboard is available for computers as well as mobile devices.

    Keyboard Layouts for Computer

    There are 3 layouts in the keyboard map: “Default” layout; “Shift” layout and “Right Alt” layout.

    Computer - Default

    The Default layout is shown below.

    1. To type the character you want, press the appropriate key. For example, A gives “ᨼ”; B gives “ᩥ” etc.
    2. The vowels ᩮ ᩯ ᩱ ᩰ and the medial consonant ᩕ are typed after the desired consonant. For example, ᨠ + ᩱ gives ᨠᩱ
    3. The ᩠ character has a special function: it allows you to create a subjoined character. Having typed the ᩠ character, just type the character you want to be subjoined. For example, ᨴ +ᩱ + ᩠ + ᨿ gives ᨴᩱ᩠ᨿ

    Computer - Shift

    The Shift part is shown below. To type the character you want, press Shift + the appropriate key (at the same time). For example, Shift + A gives ᩀ.

    Computer - Right Alt

    The Right Alt part is shown below. To type the character you want, press Right Alt + the appropriate key (at the same time). For example, Right Alt + 1 gives ᪑.
