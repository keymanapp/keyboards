CauNon Keyboard

Type Mon and Burmese Unicode in a single layout

Built specifically for the Mon community, CauNon Keyboard places Mon script and language-specific diacritics at the core of its layout. Inspired by the workflow of the classic UniMon keyboard, it seamlessly integrates Mon Unicode characters alongside standard Burmese keys. By prioritizing Mon stacked consonants and essential Burmese characters on the primary layer, CauNon keyboard eliminates the need to switch input methods when typing in both languages. The result is a faster, more intuitive typing experience that supports daily communication and language preservation.
