Allow halant ़ to be typed even after vowels: NO

Use Western digits rather than Devanagari digits by default: YES

Numbers

Numbers with backtick will get Devanagari numbers Eg: 1` will get १

Virama

X is used as virama Eg: n Xm will generate न्‌म

Conjunct characters

x is used as conjuncts Eg: n xm will generate न्म

• A simple conjunct is typed by pressing ` between any consonant. e.g: नn`n will generate न्न

• A joined conjunct is typed by pressing x between any consonant. e.g: nxn will generate न्‍न

• A non-joined conjunct is typed by pressing X between any consonants. e.g: nXn will generate न्‌न

Dependent/Independent Vowels

The keyboard will normally produce independent vowels, except after consonants, when it will produce independent vowels.

Eg:i ➔ इ vowel after consonants Eg:gi will generate गि gii will generate गी

• To force an independent vowel after a consonant, press A after the consonant. Eg: gAi will generateगइ
