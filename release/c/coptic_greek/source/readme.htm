Coptic (Greek)

This keyboard layout is designed for typing the Coptic script. This keyboard is based on Apple's Greek keyboard layout. It is useful for people who type a lot of Greek and a little Coptic.

The Antinoou font was developed by the author of this keyboard and supports the standard Coptic characters. However, it does not yet support the Coptic Epact Numbers. The Symbola font does include support for the Coptic Epact Numbers.
