Start Using chinuk_wawa

This is a keyboard for typing chinuk wawa, compatible with desktop (Windows/Mac/Linux) and mobile (Android/iOS).

For desktop layout (Windows, Mac, Linux):

Use shift to get chinuk characters:

shift + e = ə

shift + l = ɬ

shift + h = ʰ

shift + x = x̣

shift + \ = ʔ

shift + a = á

For Mobile (tablet-phone layout)

Long press on the following keys to get chinuk characters:

e + long press = ə

l + long press = ɬ

x + long press = x̣

h + long press = ʰ

. + long press = ʔ

a + long press = á
