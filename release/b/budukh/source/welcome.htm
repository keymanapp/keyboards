How to use the Budugh keyboard. The Budugh keyboard is a rota-based keyboard. That means, if you press a key for a special character more than once, you will get a different letter or combination of letters. After the final turn, the key returns to its original letter.  Additionally, the accent key produces the apostrophe symbol.

` = '

Below is a list of the characters produced by the rota

a 1x = a ; 2x = ä 3x = a

e 1x = e ; 2x = ə 3x = e

o 1x = o ; 2x = ö 3x = o

u 1x = u ; 2x = ü 3x = u

q 1x = q ; 2x = q' ; 3x = qh ; 4x = q

t 1x = t  2x = t'  3x = t

p 1x = p 2x = p' 3x = p

s 1x = s ; 2x = ts 3x = ts' 4x = ss 5x = s

g 1x = g; 2x = ğ; 3x = ˁ ; 4x = ° 5x = g

h 1x = h; 2x = ħ; 3x = ˀ ; 4x = h

k 1x = k 2x = k' 3x = k

x 1x = x 2x = xh 3x = x

c 1x = c 2x = ç 3x = ç' 4x = c

Buduq klaviaturasından necə istifadə olunur. Cari Buduq klaviaturası növbə əsaslı bir klaviaturadır. Yəni xüsusi bir simvol üçün bir düyməni bir dəfədən çox basarsanız, fərqli bir hərf və ya hərf birləşməsini alacaqsınız. Son növbədən sonra düymə orijinal hərfə qayıdır. Bundan əlavə, vurgu düyməsi basanda ayrıca apostrof simvolu cıxarır.
