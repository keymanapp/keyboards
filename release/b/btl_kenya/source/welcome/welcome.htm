Start Using Kenya (BTL)

This keyboard was developed for use by BTL (Bible Translation & Literacy) for language teams in Kenya.

This keyboard makes use of modifier keys. Typing the modifier key before the base character will output the new or combined character. For example:

;a will produce ɑ (available on aA).

'a will produce á (available on aAɑⱭeEiIoOuU).

`a will produce à (available on aAɑⱭeEiIoOuU).

:a will produce ä (available on aAɑⱭeEiIoOuU).

^i will produce î (available on iIuU).

=a will produce ā (available on aAɑⱭeEiIoOuU).

.b will produce ḅ (available on bBdD).

]d will produce d̯ (available on dDtT).

_v will produce v̱ (available on vV).
