Start Using བོད་ཡིག་གཏག་ཐབས། (Bod)

1. The input of most Tibetan characters can be achieved by directly pressing the corresponding key. For example: directly press the key where the letters ར and ག are located to enter the character རྒ.

2. When entering the letters ར, ཝ or ཡ, there is no need to consider whether they are deformed. For example, directly press the keys corresponding to the letters ར, ག and ཡ to enter the character རྒྱ. Other vertical superimposed characters (for example: ལྔ, རྒྱ, etc.) can be input similarly.

3. Press the keys corresponding to ག, ཨ and ར in sequence, you can enter གར, that is, insert the ཨ key to prevent the vertical overlap of the letters ག and ར. When the user inputs རག, གར, ལག, གལ, the user can perform similar operations.


༡. བོད་ཡིག་གི་ཡིག་འབྲུ་ཕལ་ཆེར་ཞིག་ཐེད་ཀར་དེ་དང་མཐུན་པའི་མཐེབ་བཀྱག་ལ་རྡེབ་ནས་གཏག་ཆོག དཔེར་ན། ཡིག་འབྲི་ར་དང་ག་ཡོད་སའི་མཐེབ་བཀྱག་རྡེབ་ནས་ཡི་གེ་རྒ་ཞེས་པ་གཏག་ཆོག

༢. ཡིག་འབྲུ་ར་དང་ལ་ཡང་ན་ཝ་གཏག་ན་ཁོ་ཚོས་དབྱིབས་འགྱུར་བར་བསམ་བློ་གཏོང་མི་དགོས། དཔེར་ན། ཐད་ཀར་དུ་ཡིག་འབྲུ་ར་དང་ཡ། ཝ་ཡི་གནས་མཐུན་གྱི་མཐེབ་བཀྱག་རྡེབ་ན་ཡི་གེ་རྒྱ་ཞེས་པ་གཏག་ཐུབ། བརྩེགས་ཡིག་(དཔེར་ན། ལྔ་དང་རྒྱ་སོགས་ལྟ་བུ་)གཞན་ཡང་འདི་དང་མཐུན་པར་གཏག་ཆོག

༣. ཡིག་འབྲུ་ག་དང་ཨ། ར་གསུམ་གྱི་མཐེབ་བཀྱག་རིམ་པ་ལྟར་རྡེབ་ན་ཡི་གེ་གར་ཞེས་པ་གཏག་ཆོག དེ་ནི་ཨ་མཐེབ་རྡེབ་ནས་ག་དང་ར་ཐུར་བརྩེགས་མི་བྱེད་པར་བཀག་འགོག་བྱས་པ་ཡིན། སྤྱོད་མཁན་གྱི་ཡི་གེ་རག གར། ལག གལ་ལྟ་བུ་གཏག་ཐབས་གོང་དང་མཐུན་པ་ཡིན།


1. 绝大多数藏文字符的输入可通过直接按下相应的键位来实现。例如：直接按下字母ར和ག所在的键位可输入字符རྒ。

2. 输入字母ར、ཝ或者ཡ时，不必考虑它们是否发生形变。例如，直接按下字母ར、ག和ཡ对应的键位即可输入字符རྒྱ。其它纵向叠加字符（例如：ལྔ、རྒྱ等）可类似输入。

3. 依次按下ག、ཨ和ར对应的键位，可输入གར，即通过插入ཨ键来阻止字母ག和ར的纵向叠加。用户输入རག、གར、ལག、གལ时，可类似操作。
