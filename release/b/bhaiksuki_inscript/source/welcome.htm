Start Using bhaiksuki_inscript

The Bhaiksuki script (also known as Sindhura or Arrow-Headed Script) developed during the 11th century AD. It was used to write Sanskrit in the northeast of India, and in Bangladesh, Nepal and parts of China.
This keyboard is designed for the Sanskrit language, using the Bhaiksuki script with an Inscript layout.
