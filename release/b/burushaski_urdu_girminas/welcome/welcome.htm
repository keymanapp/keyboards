Burushaski Arabic Phonetic Keyboard

By Quwat K. Sunny

This package includes a phonetic keyboard called Burushaski Girminas Urdu and the font Pak Nastaleeq optimised for use with Urdu. Please note that the On Screen Keyboard is based on a US English keyboard layout. The package inditially developed by Zeshaan Haider and Burushaski Phonetic is based on his work.
