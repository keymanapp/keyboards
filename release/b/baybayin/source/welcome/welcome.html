Basic Baybayin (ᜊᜌ᜔ᜊᜌᜒᜈ᜔) Keyboard for Philippines languages, included Tagalog, Buhid, etc. This keyboard covered all characters on the Unicode Standard 13.0. Other additional characters can be added in the future if necesarry. There is no standard keyboard layout yet for the Baybayin Keyboard at the moment, so this keyboard can be one of the alternative options.

This keyboard layout designed based on suggestions from the user group in social media. I would like to thank to the admins of Writing Script in the Philippines : John Andy Untalan Uy, Joliana Juan Sawal, Luke Biglang-Awa, Melgazar Galvan, Nathaniel Pønce de Leøn, & Raymond Bondoc Figueroa for their inputs. If there any other suggestions about the layout design, please let me know.

The font used along with this keyboard is Noto Sans Tagalog. In case you have not install it on your machine, you can simply download the font from the Google Noto website and install it.

The keyboard on this version can be installed on Windows, MacOS and Linux. It also provide the on screen keybaord..

This keybaoad is part of the Kairaga.com keyboard development program. Please send your thoughts to info@kairaga.com
