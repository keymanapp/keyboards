Bambara keyboard
==============

Description
-----------
Bamanankan (Bamanankan) sɛbɛn ka ɲɛ Windows, Mac, iOS ani Android kan. Nin klaviye in bɛ Bamanankan alfabɛ dafalen di aw ma k'a fara sigidin kɛrɛnkɛrɛnlenw kan(Ɛɛ, Ŋŋ, Ɔɔ).

Type Bambara (Bamanankan) accurately on Windows, Mac, iOS and Android. This keyboard puts the full alphabet-Including special characters(Ɛɛ, Ŋŋ, Ɔɔ).
