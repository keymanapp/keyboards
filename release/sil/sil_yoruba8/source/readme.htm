Yorùbá 8

Overview

Yorùbá 8 is a simple Unicode keyboard for the Yorùbá language, with the option to use either the modern dot or the traditional bar under the letters ẹ|e̩, ọ|o̩, and ṣ|s̩.

Basic Function

The Yorùbá 8 keyboard adapts to your default Windows layout. You can type all of the special Yorùbá letters and accents with Yorùbá 8. For example, type q for gb or [ for ` (grave accent). Accents are typed after the base letter and can be added to vowels as well as m and n. When the keyboard is on, type Ctrl+. to switch between the dot form and the bar form of the letters ẹ|e̩, ọ|o̩, and ṣ|s̩. Keyman Desktop remembers your choice whenever you return to use the keyboard.

Note: For best practice, we do not recommend using both the dot and the bar forms of the Yorùbá letters in the same document.

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Yorùbá.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard displays best on a QWERTY US layout.
