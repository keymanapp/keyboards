Start Using Yorùbá 8

Overview

Yorùbá 8 is a simple Unicode keyboard for the Yorùbá language, with the option to use either the modern dot or the traditional bar under the letters ẹ|e̩, ọ|o̩, and ṣ|s̩.

Function

The Yorùbá 8 keyboard is adaptive. The arrangement of keys in Yorùbá 8 will adapt to your default Windows layout. For example, you can find the ´ (acute accent) on the ] key on your keyboard, no matter whether you are using an English(QWERTY), a German (QWERTZ), or a French (AZERTY) keyboard.

You can type all of the special Yorùbá letters with a single key using this keyboard. Here is how:

Key Accent
q gb
z ṣ or s̩
x ọ or o̩
v ẹ or e̩
^ ₦

You can chose with this keyboard whether to use the traditional bar form or the modern dot form of the letters ẹ|e̩, ọ|o̩, and ṣ|s̩. When the keyboard is on, type Ctrl+. to switch between the dot form and the bar form of the letters. Keyman Desktop remembers your choice whenever you return to use the keyboard.

Note: For best practice, we do not recommend using both the dot and the bar forms of the Yorùbá letters in the same document.

You can type all of the Yorùbá accents with this keyboard. Here is how:

Key Accent
[ `
] ´
{ ˆ
} ˇ
| ˉ

Accents are typed after the base leter. Accents can be added to the letters a, e, ẹ or e̩, i, m, n, o, ọ or o̩, and u.

Example: type a[ to get à.

Example: type M| to get M̄.

To get the standard output for any key instead of Yorùbá, type the ` first. This is shown as a ★ on the On Screen Keyboard.

Example: type `Q to get Q (instead of GB).

Example: type `{ to get { (instead of ˆ).

Example: type `v to get v (instead of ẹ or e̩).

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Yorùbá.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard displays best on a QWERTY US layout.
