Yorùbá 8 keyboard
==============

(c) 2012-2019 SIL International

Version 1.1

Description
-----------

Yorùbá 8 is a simple Unicode keyboard for the Yorùbá language, with the option to use either the 
modern dot or the traditional bar under the letters ẹ|e̩, ọ|o̩, and ṣ|s̩.
