Cameroon QWERTY Usage

This keyboard supports the Standardized Alphabet for Cameroonian Languages for a QWERTY (US Keyboard).

Keyboard Information

This chart describes the keystrokes used in Cameroon QWERTY keyboard layout.

Use the ; (semicolon) key for the Cam key and the : (colon) key for the special (Spcl) key.

A non-combining version of any tone/diacritic marker may be inserted by itself into a text by typing the desired tone or diacritic key, followed by +. Note that this feature is not available for the four complex tones, dot-under, or tilde-under.

When adding nasal or tone diacritics, the typing sequence is always: Nasal ( ◌̃ ◌̧ ) + Tone ( ◌̀ ◌́ ◌̂ ◌̄ etc.) + Vowel.
