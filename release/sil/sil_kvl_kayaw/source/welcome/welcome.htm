The SIL Kayaw Keyman Keyboard Package

Keyboard

This keyboard layout is designed for Kayaw. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item. This keyboard follows the standard QWERTY keyboard layout the following changes:

* q is replaced by u0304 (Combining Macron) * z is replaced by u0300 (Combining Grave Accent) * x is replaced by u0302 (Combining Circumflex Accent) * v is replaced by u030C (Combining Caron)
