Taang (SIL)

This keyboard is designed for Rumai Palaung language spoken in Myanmar and many other countries.

It is based on Taang Unicode.

Desktop Keyboard Layout

Default

Shift

Mobile Keyboard Layout

Press the top left key (dotted circle) to combine consonants.

Longpress on the top row of keys on the default layer to get numbers.

Several punctuation characters are on the longpress of the "." key.

Default

Shift
