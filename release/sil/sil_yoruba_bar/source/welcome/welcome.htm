Overview

Yorùbá with Bar is a simple Unicode keyboard for the Yorùbá language, using the traditional bar under the letters e̩, o̩, and s̩.

Function

The Yorùbá with Bar keyboard follows an English (QWERTY US) layout. You can type all of the special Yorùbá letters with a single key using this keyboard. Here is how:

Key Accent
q gb
z s̩
x o̩
v e̩
^ ₦

You can type all of the Yorùbá accents with this keyboard. Here is how:

Key Accent
[ `
] ´
{ ˆ
} ˇ
| ˉ

Accents are typed after the base leter. Accents can be added to the letters a, e, e̩, i, m, n, o, o̩, and u.

Example: type a[ to get à.

Example: type M| to get M̄.

To get the standard output for any key instead of Yorùbá, type the ` first. This is shown as a ★ on the On Screen Keyboard.

Example: type `Q to get Q (instead of GB).

Example: type `{ to get { (instead of ˆ).

Example: type `v to get v (instead of e̩).

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Yorùbá.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard displays best on a QWERTY US layout.
