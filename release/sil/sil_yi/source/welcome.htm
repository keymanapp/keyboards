SIL Yi keyboard - Mobile & Desktop

Mobile and desktop keyboards type Nuosu Yi syllables, Yi radicals, and full-width punctuation in Unicode encoding, viewable with a font such as Nuosu SIL.

Mobile

On mobile platforms, choose a Yi Pinyin consonant then vowel+tone. Syllables that begin with a vowel have null (Ø) consonant. This method lets you type any Yi syllable character with just two key strokes.

Consonant Vowel+Tone Syllable
Ø iet ꀄ
nr a ꎖ

In the default layer and the numeric layer, the backspace key removes the last code point that was typed. In other layers, the backspace key returns to the default layer without typing a character.

To type space, touch the #.? key to go to the numeric layer, then type the key in the upper left corner. (It is labeled "sil_yi" because Keyman automatically puts the keyboard name on the space bar.)

To type a Yi radical, long press the #.? key, then touch a Yi radical. To type a variant form of a radical, long press the base form of the radical.

Desktop

Type Latin letters to spell a syllable in Yi Pinyin, then space to enter the syllable character. Type "w" + space to enter the syllable iteration mark ꀕ. To type a Yi radical, append "b" (for ꀨ) to its Yi Pinyin name.

Type Yi Pinyin letters Display
i + e + t + space ꀄ
n + r + a + space ꎖ
w + space ꀕ
z + o + t + b + space ꒫

Typing punctuation

Note the keystrokes at the bottom of the list for typing book title brackets and left and right Chinese quotation marks. For instance, typing two left angle brackets in a row will produce a Chinese left double book title bracket. Typing a backslash produces a Chinese dunhao. Typing two left single quotes in a row produces a full-width left double quotation mark.
