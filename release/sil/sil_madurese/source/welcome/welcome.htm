Start Using sil_madurese

A keyboard for Madurese language written with Latin script

Keyboard Layout

Special key combinations to use on desktop computers:

  • `e → è
  • `E → È
  • /a → â
  • /A → Â
  • .d → ḍ
  • .D → Ḍ
  • .t → ṭ
  • .T → Ṭ

Type the modifer keys twice to output:

  • `` → `
  • // → /
  • .. → .
