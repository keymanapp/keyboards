Uganda-Tanzania Bantu (SIL) Keyboard

Keyboard © 2004-2019 SIL International

Overview

This keyboard is intended to support all the Bantu languages of Uganda and Tanzania.

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for extended Latin characters. To find fonts on your computer which support the keyboard, use the Keyman Desktop Font Helper.

SIL International has produced several Unicode font sets that support all the characters in this keyboard. These fonts are:

    Charis SIL
    Doulos SIL
    Andika
