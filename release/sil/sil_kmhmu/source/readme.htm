Kmhmu (SIL) Keyboard (Lao orthography)

This package includes the Kmhmu (SIL) keyboard and a compatible font Kmhmu MX and Kmhmu OT.

Keyboard layout follows the Lao standard, with some minor differences to simplify entry of Kmhmu text.

See the keyboard instructions or the On Screen Keyboard for layout details.

Note that not all standard Lao Unicode (OpenType) fonts permit all character sequences found in Kmhmu.
