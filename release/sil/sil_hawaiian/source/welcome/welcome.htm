Start Using Hawaiian (SIL)

Hawaiian (SIL) is a Unicode keyboard with compatible fonts, included with permission from Hale Kuamo'o.

Keyboard Layout

Default (unshifted)

Shift

Right ALT (unshifted)

Shift Right ALT

Fonts

HI Kakuhihewa Unicode, HI Keawe Unicode, HI Manokalanipo Unicode and HI Pi'ilani Unicode fonts have been installed with this keyboard package and are recommended for use with Hawaiian.

These fonts are all Copyright © 2002 Hale Kuamo'o and are distributed with this keyboard with permission from Hale Kuamo'o.

Keyboard Layout Notes

KeyOutput
RightAlt+aā
RightAlt+eē
RightAlt+iī
RightAlt+oō
RightAlt+uū
RightAlt+'ʻ

You can access the capital letter versions of these vowels with Caps Lock or Shift.

ANSI Font Notes

This package does not include fonts for the Hawaiian ANSI keyboard ("HI Fonts"). ANSI fonts can be downloaded from http://www.olelo.hawaii.edu/
