Start Using Tanguat (SIL)

This keyboard is designed for the Tanguat language of PNG

To type ñ, use c.

To type ə, use f.

To type Ñ, use C.

To type Ə, use F.

To type c, use ].

To type f, use [.

To type C, use }.

To type F, use {.

To type [, use right-alt [.

To type [, use right-alt ].

To type {, use shift right-alt { .

To type }, use shift right-alt }.
