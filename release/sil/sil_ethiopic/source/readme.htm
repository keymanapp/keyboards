SIL Ethiopic Keyboard

Introduction

The SIL Ethiopic keyboard enables typing in all Ethiopic Saba Fidel scripts. You can use this keyboard with any Ethiopic Unicode font unless you need to use the newer Unicode 6.0+ Ethiopic codepoints (for Basketo, Gumuz and Gamo-Gofa-Dawro) or the two Private Use Area (PUA) characters of this keyboard. In these cases, the Abyssinica SIL font (version 1.5+) is recommended.
