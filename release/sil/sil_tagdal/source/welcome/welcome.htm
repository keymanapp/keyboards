Tagdal (SIL) Keyboard Help

Tagdal keyboard with Latin script and some IPA characters accessible on the Right-Alt layer.

Key features:
  1. Press ] to add combining dot below
  2. Access some IPA charactes with AltGr (Right Alt)

Keyboard Layout

Desktop - Default layer

Desktop - Shift layer

Desktop - Right-Alt layer

Touch - Default layer

Touch - Shift layer

Touch - RightAlt layer

Touch - Numeric layer

Touch - Longpress key

Any key with a little gray dot on the top right can be pressed and held to reveal more key(s) as shown below.
