SIL Sahu Keyboard for Keyman

The SIL Sahu Keyman keyboard is a mnemonic keyboard definition which enters Unicode values for the Sahu language. The Desktop version of the keyboard supports a mnemonic layout which allows a user to type with any kind of physical keyboard, rather than just a US keyboard.

Fonts

This keyboard uses Charis SIL. This package will install Charis SIL for you.
