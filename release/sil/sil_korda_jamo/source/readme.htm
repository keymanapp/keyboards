Korean KORDA Jamo (SIL) Keyboards Help

The Korean KORDA Jamo (SIL) keyboard is built to type Korean using the transliteration system KORDA (Korean Romanization for Data Application). KORDA is designed to make Latin-Korean transliteration easy, accurate, and computerizable. For more information on KORDA, see About KORDA.

The Korean KORDA Jamo (SIL) keyboard creates hangeul syllables from jamo letters each time you press Spacebar.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard works best when associated with a QWERTY US layout.

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Korean characters. Common fonts which work with this keyboard are:

  • Arial Unicode MS
  • Batang
  • Dotum
  • Gulim
  • Gungsuh
  • Malgun Gothic
