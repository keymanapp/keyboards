Overview

Yorùbá with Dot is a simple Unicode keyboard for the Yorùbá language, using the modern dot under the letters ẹ, ọ, and ṣ.

Basic Function

The Yorùbá with Dot keyboard adapts to your default Windows layout. You can type all of the special Yorùbá letters and accents with Yorùbá with Dot. For example, type q for gb or [ for ` (grave accent). Accents are typed after the base letter and can be added to vowels as well as m and n.

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Yorùbá.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard displays best on a QWERTY US layout.
