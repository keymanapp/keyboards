Start Using Indic Roman Transliteration (SIL)

This is a simplified keyboard for typing Roman script with accents or diacritics

This package includes the font Charis SIL. You can use this font to guarantee that all the diacritics are correctly displayed.

The plus (+) means these keystrokes are typed successively. For example, "^ + a" means you would type "^" followed by "a" to get "â".

As this keyboard is intended for transliteration, combining marks on consonants are not supported.
