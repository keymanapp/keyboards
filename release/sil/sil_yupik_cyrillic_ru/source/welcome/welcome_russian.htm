Эскимосская клавиатура (русская раскладка)

Разработка - SIL International

English

Эскимосская клавиатура (русская раскладка) позволяет пользователю печатать кириллическим шрифтом, используя раскладку, основанную на стандартной русской клавиатуре. Русская раскладка была изменена, чтобы предоставить доступ к дополнительным символам, необходимым для эскимосского языка. Клавиатура теперь доступна как для настольного компьютера, так и для сенсорного экрана.

Требования к шрифту

Клавиатуру можно использовать с любым шрифтом, поддерживающим:

  • блок Юникода кириллического алфавита (U+0400..U+04FF) и
  • использование комбинирования диакритических знаков. (В эскимоской прописи используется макрон над некоторыми гласными.)

Никакие шрифты не включены в этот пакет клавиатуры.

Раскладка клавиатуры для настольного компьютера

На следующих диаграммах клавиатуры любую клавишу, обозначенную синим символом, за которыми следует красный символ, можно нажать один раз, чтобы получить синий символ, или дважды, чтобы получить красный символ. Например, нажатие

 
я я̄

один раз выведет “я”, нажатие на него во второй раз заменит “я” на “я̄”.

Раскладка клавиатуры для сенсорного экрана

Раскладка клавиатуры для сенсорного экрана очень похожа на раскладку для настольного компьютера. Несколько символов ( ё Ё = + ъ Ъ \ / . , ) в углах настольного компьютера пришлось переместить. Доступ к символам эскимосского языка можно получить либо двойным нажатием клавиши, либо длительным нажатием клавиши.

На приведенной ниже диаграмме показаны как символы по умолчанию для каждой клавиши, так и любые символы длительного нажатия, связанные с клавишей. Каждая клавиша разделена на две части. Большой символ в нижнем поле является символом по умолчанию для клавиши — это то, что вы получите, если просто нажмете клавишу обычным способом. Маленькие символы в верхнем поле - это символы длительного нажатия, связанные с клавишей. Если клавиша не содержит символов длительного нажатия, верхний флажок отображается серым цветом. Например:

 
Қ 4
к
← символы длительного нажатия
← символ по умолчанию


 
Ф
← нет символов длительного нажатия
← символ по умолчанию
