The Yupik Cyrillic keyboard (sil_yupik_cyrillic_ru) is now available in both desktop and touch screen layouts.

Desktop Keyboard Layout

The desktop version is based on a standard Russian layout. It has been modified to give access to the additional characters required for the Yupik language, as described by the following charts.

Yellow keys—press any of these keys twice to generate the corresponding Yupik consonant with descender: Қ Ң Ӷ Ҳ. Pressing 'В' twice will generate 'Ў'.

Orange keys—press any of these keys twice to generate the corresponding Yupik vowel with macron above it: Ӯ А̄ Я̄ Ӣ Ю̄

Touch Screen Layout

The touch screen layout is very similar to the desktop layout. Several characters (ёЁ=+ъЪ\/.,) at the corners of the desktop layout had to be relocated. Yupik characters can be accessed using either a double key press or a long key press.

Keys with a diagonal line in the upper right corner—The diagonal line marks the keys that will respond to a long press. If the key is also colored, at least on of the long-press characters will be a character used in Yupik. Otherwise, the long press will contain numbers, punctuation, or alphabetic characters that are not used in Yupik.

Yellow keys—a double key press or long key press will generate the corresponding Yupik consonant with descender: Қ Ң Ӷ Ҳ. A double or long key press on 'В' will generate 'Ў', and on 'Ь' will generate the hard sign 'Ъ'.

Orange keys—a double or long key press will generate the corresponding Yupik vowel with macron above it: Ӯ А̄ Я̄ Ӣ Ю̄
