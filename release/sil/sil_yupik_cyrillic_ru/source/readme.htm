Yupik Cyrillic RU (SIL)

SIL's Yupik Cyrillic RU keyboard has been designed for typing the Central Siberian Yupik language in the Cyrillic script (ess-Cyrl). It is based on the layout of the Russian Basic keyboard, but additional characters required by the Yupik language have been added.
