The SIL Boonkit Keyman Keyboard Package

Font

The "SIL Boonkit" keyboard is designed to work with the Tai Tham (Lanna) script.

Keyboard

This keyboard layout is designed for Lanna. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.
