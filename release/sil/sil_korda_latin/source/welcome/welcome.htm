Start Using Korean KORDA Latin (SIL) Keyboard

Overview

The Korean KORDA Latin (SIL) keyboard is built to type Korean using the transliteration system KORDA (Korean Romanization for Data Application). KORDA is designed to make Latin-Korean transliteration easy, accurate, and computerizable. For more information on KORDA, see About KORDA.

The Korean KORDA Latin (SIL) keyboard creates hangeul syllables from jamo or latin letters each time you press Spacebar.

On Screen Keyboard

This keyboard includes an On Screen Keyboard view for easy reference. The On Screen Keyboard works best when associated with a QWERTY US layout.

Fonts

This is a Unicode keyboard and works with any Unicode font which has support for Korean characters. Common fonts which work with this keyboard are:

  • Arial Unicode MS
  • Batang
  • Dotum
  • Gulim
  • Gungsuh
  • Malgun Gothic

Keyboard Details

Korean KORDA Latin (SIL) Keyboard

The Korean KORDA Latin (SIL) keyboard converts Latin letters into hangeul after you press Spacebar. As you type, Latin letters are shown. When you press Spacebar, the letters are converted into a hangeul syllable.

Example: type sam to see sam. Press Spacebar to get 삼.

Pressing Spacebar twice will insert a space.

The letters c, y, and w require you to type another key to get the right jamo for your hangeul syllable. The letter c needs h. The letters y and w need vowels.
