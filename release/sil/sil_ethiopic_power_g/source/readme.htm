Introduction

The SIL Ethiopic Power-G keyboard was developed to input Ethiopic Saba Fidel scripts using the keyboarding sequences found in the “Power Ge'ez (Phonetic)” keyboard from Concepts Data Systems. You can use this keyboard with any Ethiopic Unicode font unless you need to use the newer Unicode 6.0+ Ethiopic codepoints (for Basketo, Gumuz and Gamo-Gofa-Dawro) or the Private Use Area (PUA) characters of this keyboard. In those cases, the Abyssinica SIL font (version 1.5+) is recommended.
