Clavier mooré

Ce clavier est destiné à la saisie et à l’affichage de textes/documents en langue mooré. Il est construit à partir du clavier AZERTY français. Une infographie incluse dans ce pack permet de repérer les différents emplacements des caractères spécifiques au mooré.

Mooré Keyboard Layout

This keyboard is designed for input of the Mooré language. It is based on the French keyboard as used in Burkina Faso. A visual keyboard is included in this package; please refer to it for placement of specific characters.
