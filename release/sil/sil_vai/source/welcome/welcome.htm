Vai (SIL) Keyboard

Introduction

This keyboard was developed for keyboarding the Vai Script. It is based on the old custom encoded keyboard, but it has been updated for Unicode. The majority of the keystrokes remain the same. (One significant difference is that an “x” is used instead of an “r” for the “o” forms).

The SIL keyboard is designed to work with any Unicode Vai font.

Keyboard assignments

In the chart below, what is in the keystroke column must be typed first followed by the “vowel” in the top of the corresponding column. For example, type “pe” for ꔅ or “bhq” for ꗩ.
