The Torwali Keyman Keyboard Package

Font

The Torwali Keyman keyboard is designed to work with the Scheherazade font. However, you can use this keyboard with any extended Arabic script font. Unicode font.

Keyboard

This keyboard layout is designed for the Torwali language. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.

Keyboard Layout
