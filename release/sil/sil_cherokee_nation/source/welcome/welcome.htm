Start Using Cherokee Nation (SIL)

A Cherokee Unicode keyboard following the official Cherokee Nation layout. A font must be installed to support this keyboard. This keyboard comes with the Aboriginal Serif font which supports the Cherokee script. Windows includes a number of fonts which support the Cherokee script including the Gadugi and Plantagenet Cherokee fonts.

However, you can use this keyboard with any Cherokee script Unicode font.
