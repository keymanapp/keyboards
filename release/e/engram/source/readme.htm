Arno's Engram key layout

The Engram layout (website and GitHub repo) is a keyboard layout optimized for comfortable and efficient touch typing in English created by Arno Klein, with open source code to create other optimized key layouts. Letters are optimally arranged to reduce lateral finger movements, following ergonomic principles, and facilitating high-frequency letter pairs. Punctuation marks are grouped together in the middle columns (accessed by the Shift and R-Alt keys) and numbers are paired with symbols that resemble them (accessed by the Shift Key):
