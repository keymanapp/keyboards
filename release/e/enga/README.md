Enga keyboard
=============

© Copyright 2019 Stanley Stanis Kaka, Marc Durdin

Version 1.0

Description
-----------

The Enga keyboard supports the Enga language of Papua New Guinea. It includes four characters for Enga language: ē ŋ Ē Ŋ.

Supported Platforms
-------------------
 * Windows
 * macOS
 * Linux
 * Web
 * iPhone
 * iPad
 * Android phone
 * Android tablet
 * Mobile devices
 * Desktop devices
 * Tablet devices
