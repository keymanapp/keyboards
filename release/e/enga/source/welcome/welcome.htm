Start Using Enga

The Enga Keyboard includes 4 characters for the Enga language of Papua New Guinea: ē ŋ Ē Ŋ.

On hardware keyboards, the keyboard overrides c for ē and f for ŋ, as c and f are not used in Enga. To access c and f (e.g. for loan words), use Right Alt.
