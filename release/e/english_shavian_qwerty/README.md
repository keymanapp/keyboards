Shaw QWERTY keyboard
==============

Description
-----------

Shavian alphabet keyboard for English, inspired by the old pre-Unicode, QWERTY-based ASCII mapping commonly used for Shavian.

Links
-----
https://www.shavian.info/keyboards/

Supported Platforms
-------------------

 * macOS
 * Linux
 * Web
 * iPhone
 * iPad
 * Android phone
 * Android tablet
 * Mobile devices
 * Desktop devices
 * Tablet devices
