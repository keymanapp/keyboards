Aksara Bali Unicode Keyboard

Keyboard © 2019 The PanLex Project of The Long Now Foundation

This keyboard lets you type Askara Bali (Balinese script) in Unicode. It is based on the Bali Simbar non-Unicode layout.

Please read the Bali Simbar keyboard guide (in Indonesian) for instructions. This keyboard follows the Bali Simbar keyboard as closely as possible, and supports the following additional keystrokes:

  • Ctrl-r produces rerekan, for example: [p] [Ctrl-r] produces [ᬧ᬴]
  • Ctrl-Alt-o produces windu: [᭜]
  • Ctrl-Shift-, produces [᭚᭜᭚], repeat for [᭟᭜᭟]
  • Ctrl-Shift-. produces [᭛᭜᭛], repeat for [᭟᭜᭟]
  • Ctrl-Shift-/ produces [/]
