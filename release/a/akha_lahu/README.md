Akha/Lahu keyboard
==============

Description
-----------
This is a keyboard that contains all of the characters needed to type the Akha and Lahu languages. Although it is Unicode, the keyboard follows the legacy Akha ACT keyboard layout. The keyboard produces NFD Unicode.
