About ausephon


We Use This Daily 

The ausephon keyboards are a work in progress. So far we have designed a keyboard for phonemic (broad) transcription of Australian accented English speech. Sometime in the future, we will release a phonetic (narrow) keyboard.  

We find the phonemic keyboard useful for quickly touch typing symbols and becoming  fluent with transcribing the HCE (Harrington/Cox/Evans) system.

Key Sequences


The SHIFT key

Pressing the shift key (in yellow), and then one of those white keys in the layout (below) provides access to the symbols shown in the top right-hand area of each key. e.g. the shift key, then the 'e' key will output schwa /ə/.


Diacritics

To type diacritics you will first need to type the letter to which you want to add a diacritic. For example, to type the consonant marker below 'l', 'm', 'n', or 'r', type one of these aforementioned letters and then the BACKTICK key (in yellow). This will produce the syllabic consonant marker below the character: l̩, m̩, n̩, ɹ̩.
