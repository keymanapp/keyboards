Arabic Flick keyboard
==============

Description
-----------
Flick style keyboard for Arabic, like the T9 Style keyboard for Japanese. In main row, every key can be flicked to all four sides, and also multitapped just like the old T9 Phone keyboards. In number row, Arabic number are main press, and Hindi numerals are swipe-up, but not multitap, as numbers can be repeated.
