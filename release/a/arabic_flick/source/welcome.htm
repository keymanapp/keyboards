Start Using Arabic Flick

Flick style keyboard for Arabic, like the T9 Style Flick keyboard for Japanese.

Keyboard Layout

Main press is the number you see on main. Swipe to get the number written in hints, and there order is as follows: Right, Top, Left, Down. Similarly, can also be long pressed in to get the characters in the order they are written in key hints.The arrows are to move, right and left. Symbols key leads you multi-layer symbol row, while number key leads you to T9 style number row. The shadda below can be longpressed, or multitapped, or flicked for more diacritics.

The main numbers are Arabic Numerals, and Swipe-Up are Hindi Numerals. Double tapping on numbers doesn't change the letters, as numbers are repeated many times.
