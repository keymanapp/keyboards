ጉራጊና (Gurage) and አማርኛ (Amharic) Keyboard Package

This package includes 2 mnemonic keyboards called: ጉራጊና (Gurage) and አማርኛ (Amharic).

The Gurage (ጉራጊና , ISO-639-2 swg) keyboard supports the orthography devised by Dr. Fekede Menuta and adopted in Unicode 14. It requires a font supporting the Ethiopic Extended-B block defined in the Unicode 14 standard.

The Amharic (amh, አማርኛ) keyboard applies Ethiopian writing conventions. It requires a font supporting Ethiopic script under the Unicode 4.1 standard.
