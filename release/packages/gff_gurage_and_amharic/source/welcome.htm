ጉራጌ (Gurage) and አማርኛ (Amharic) Keyboard Package

This package includes 2 mnemonic keyboards called: ጉራጌ (Gurage) and አማርኛ (Amharic).

The Gurage (ጉራጌ, ISO-639-2 sgw) keyboard supports the experimental orthography devised by Dr. Fekede Menuta. It requires the corresponding Zebidar font provided with this package.

The Amharic (amh, አማርኛ) keyboard applies Ethiopian writing conventions. It requires a font supporting Ethiopic script under the Unicode 4.1 standard.
