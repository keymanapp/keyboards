FirstVoices Keyboard Package

(c) 2015-2026 FirstVoices, SIL Global, 2015 First Peoples' Cultural Foundation

This package includes all the FirstVoices keyboards for desktop PCs and mobile. It is distributed as part of the FirstVoices project.

Keyboards include:

  • Canadian Multilingual Standard Basic
  • EuroLatin (SIL)
  • Inuktitut keyboards
  • FirstVoices keyboards
