Keyboard Layout

Typing Nasal Vowels

To type nasal vowels, press the tilde key "~" and then type "a, e, i, or u". The tilde should appear directly above the vowel.



Typing Long Vowels

To type long vowels, press the backtick key "`" and then type "a, e, i, or u". The accent should appear directly above the vowel.
