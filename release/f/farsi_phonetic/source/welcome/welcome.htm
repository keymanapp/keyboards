Start Using Farsi Phonetic

Persian phonetic keyboard which also supports related languages.

The standard "\" is a deadkey. This can be typed preceding a space or any of the following characters: space, zero, waw, heh, fatha, alef, guillemet open, guillemet closed. The resulting characters will be backslash, latin zero, alef with hamza, fathatan, heh with yeh above, waw with hamza above, double guillemet open, double guillemet closed.
