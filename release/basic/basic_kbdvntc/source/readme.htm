This keyboard defines some keys that are set by default as "hotkeys" when Keyman is installed. To use this keyboard, you will want to go to Configuration.../Hotkeys, select the hotkey name, and either change the hotkey or turn it off. Currently "Switch Keyman Desktop Off" uses Shift+Alt+O "Open Keyboard Menu" uses Shift+Alt+M "Show On Screen Keyboard Pane" uses Shift+Alt+.K
