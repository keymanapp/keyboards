This keyboard makes use of deadkeys.

To get "ā", type "RALT - a". This is available on a, e, i, o, and u.

To get "ê", type "SHIFT RALT ^ e". This is available on e and o.

To get "ĕ", type "SHIFT RALT ( e". This is available on e, h, o, and u.

To get "ḍ", type "RALT . d". This is available on d, h, l, m, n, r, s, and t.

To get "ḏ", type "SHIFT RALT _ d". This is available on d, h, k, l, n, r, and t.

To get "ñ", type "SHIFT RALT ~ n"

To get "ž", type "SHIFT RALT < z". This is only available on z.

A number of other combining marks are available on deadkeys. Typing a letter + RALT + comma with any of the following: ^*9(o-.:_>`~|' can be used to produce the following: âa̐a̮ăḁāạa̤a̱ȧàãa̲á
