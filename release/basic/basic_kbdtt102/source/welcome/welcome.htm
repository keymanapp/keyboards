This keyboard makes use of SHIFT+RALT+O and SHIFT+RALT+M keys which are set by default when Keyman is installed. To use this keyboard, you must go to Configuration.../Hotkeys and select the hotkey for "Switch Keyman Desktop Off". Either change the hotkey or turn it off. Do the same for "Open Keyboard Menu" and for "Open Language Switcher".
