Start Using Icelandic Basic

This keyboard layout is designed for Icelandic. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item. The keyboard layout follows the Windows 10 Icelandic layout.

This keyboard makes use of "deadkeys" which are listed below:

`a will produce à (available on aAeEiIoOuU).

°a will produce å (available on aA).

¨a will produce ä (available on aAeEiIoOuUy).

´a will produce á (available on aAeEiIoOuUyY).

^a will produce â (available on aAeEiIoOuU).
