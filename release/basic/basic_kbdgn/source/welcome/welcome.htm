Keyboard Layout

This keyboard makes use of SHIFT+RALT keys which are set by default when Keyman is installed. To use this keyboard, you must go to Configuration.../Hotkeys and select the hotkey for "Switch Keyman Desktop Off". Either change the hotkey or turn it off. Do the same for "Open Keyboard Menu" and for "Open Language Switcher".

Deadkeys are used to output vowels with accent marks.

  • Type ◌́ followed by any vowel (aeiouyAEIOUY) to output an acute accent on the vowel.
  • Type ◌̈ followed by any vowel (aeiouyAEIOU) to output a diaeresis/umlaut accent on the vowel.
  • Type ◌̂ followed by any vowel (aeiouAEIOU) to output a circumflex accent on the vowel.
  • Type ◌̃ followed by any vowel (aeiouyAEIOUY) to output a tilde accent on the vowel.
  • Type ◌̀ followed by any vowel (aeiouAEIOU) to output a tilde accent on the vowel.
