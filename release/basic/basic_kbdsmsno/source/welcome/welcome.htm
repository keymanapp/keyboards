Start Using Sami Extended Norway Basic

This keyboard layout is designed for Sami (Northern). It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item. The keyboard layout follows the Windows 10 Sami Extended Norway layout.
