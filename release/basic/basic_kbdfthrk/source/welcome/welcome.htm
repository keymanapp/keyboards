A font must be installed to support this keyboard. Windows includes the Segoe UI Historic font which supports the Runic script.

The following characters can also be used with the "\" and "|" deadkeys: ᚬᛒᚼᛅᛘᛦᛋ. For example, you can type "\" + "ᛅ" to output "ᛆ".

If "\" followed by a space is typed, a "~" will be output. If "|" followed by a space is typed, a "|" will be output.
