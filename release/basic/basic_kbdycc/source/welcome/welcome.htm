Start Using Serbian (Cyrillic) Basic

This keyboard layout is based on the Serbian (Cyrillic) Basic keyboard layout from Windows 10.

This keyboard makes use of deadkeys. Typing the deadkey before the base character will output the combined character. For example:

'г will produce ѓ (available on гГкК).
