Start Using Inuktitut - Naqittaut Basic

This keyboard layout is designed for Inuktitut - Naqittaut (Latin and Syllabics). It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item. The keyboard layout follows the Windows 10 Inuktitut - Naqittaut layout.

This keyboard uses one deadkey "]" which is used with the Naquittaut characters to access alternate Naquittaut characters. The Naquittaut characters may also be accessed through the RALT and SHIFT+RALT keys.

This keyboard defines some keys that are set by default as "hotkeys" when older versions (pre ver. 13) of Keyman is installed. To use this keyboard, you will want to go to Configuration.../Hotkeys, select the hotkey name, and either change the hotkey or turn it off. Currently "Switch Keyman Desktop Off" uses Shift+Alt+O "Open Keyboard Menu" uses Shift+Alt+M "Show On Screen Keyboard Pane" uses Shift+Alt+K
