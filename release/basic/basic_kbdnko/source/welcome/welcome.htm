Start Using N’Ko Basic

This keyboard is designed for languages using the N’Ko script.

A font must be installed to support this keyboard. Windows includes the Ebrima font which supports the N'Ko script.

The Noto Sans NKo font is included in the keyboard package to avoid tofu of the newly added characters.
