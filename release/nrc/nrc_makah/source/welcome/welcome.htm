Welcome to the Makah (Qʷi·qʷi·diččaq) keyboard

This package contains both a mobile keyboard layout (for smartphones and tablets) and a desktop keyboard layout (for Windows, macOS, and other physical keyboards).

Mobile keyboard

Tutorial

This is a longpress keyboard. To type letters like č, c̓, č̓, press and hold the most similar letter: in this case, c.

Typing English R, F, V

The keys for English R, F, V are replaced with common Makah letters on this keyboard, but you can still type the English letters. Longpress the key where the letter would ordinarily be on an English keyboard.

Desktop layout

The desktop layout implements the layout designed in collaboration with the Makah Language Program, itself based on an older Language Geek keyboard.

Please see the Makah Keyboard page on the Makah Museum for more information about this keyboard layout.

Keyboard layout

To get the bottom-right character (in blue), hold the Alt key on the right-hand side of the keyboard, then press the desired key.

To get the top-right character (in green), hold any Shift key, in addition to Alt on the right-hand side of the keyboard, then press the desired key.

To add a glottal mark to a letter (for example, k → k̓), press the desired constant key first, then press the ◌̓ key.
