ᓀᐦᐃᔭᐍᐏᐣ (Cree Syllabics)

This keyboard is for writing nêhiyawêwin/Plains Cree/Cree Y-dialect in syllabics (ᒑᐦᑭᐯᐦᐃᑲᐣ) on mobile devices (e.g., iPhone/Android phones).

This keyboard was created by Eddie Antonio Santos of the National Research Council Canada's Canadian Indigenous languages technology project.
