Start using the Xinaliq Keyboard

Keyboard for Xinaliq (kjj) using the Latin Orthography approved by the Ministry of Education of the Republic of Azerbaijan.

Desktop Keyboard layout

This keyboard uses a rota to access alternate characters based on the default shape. To get "g", you type "g", to get "ğ" type "gg".

Characters that use the rota system are c, e, g, h, i, k, o, p, q, s, u, t, x, z (and their uppercase counterparts).

Default (unshifted)

Voiceless Stops

q: 1x = q, 2x = q̇ ,3x = q̂, 4x = q̂q̂, 5x = q (returns to the base character)

t: 1x = t, 2x = ṫ, 3x = t̂, 4x = t̂t̂, 5x = t

p: 1x = p, 2x = ṗ, 3x = p̂, 4x = p̂p̂, 5x = p

k: 1x = k, 2x = k̇ ,3x = k̂, 4x = k̂k̂, 5x = k

Other Consonants

s: 1x = s, 2x = ŝ, 3x = ş, 4x = s (returns to base character)

c: 1x = c, 2x = ċ, 3x = ĉĉ, 4x = ç, 5x = c

g: 1x = g, 2x = ğ, 3x = ğg, 4x = ĝ, 5x = g

h: 1x = h, 2x = ḣ, 3x = ĥ, 4x = h

z: 1x = z, 2x = ż, 3x = ẑ, 4x = z

x: 1x = x, 2x = x̂, 3x = x

Vowels

e: 1x = e, 2x = ə, 3x = e

u: 1x = u, 2x = ü, 3x = u

o: 1x = o, 2x = ö, 3x = o
