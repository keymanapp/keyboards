Start Using Dzardzongke Keyboard

This keyboard types the Dzardzongke language spoken in Nepal, using the Dzardzongke orthography developed by Marieke Meelen and Charles Ramble.

Keyboard Layout

The keyboard layout is based on that of a standard English keyboard, and uses ; to type special characters. See this document for all the key combinations.

Predictive Texting

For mobile devices, you can also install a 'lexical model' to enable predictive texting in Dzardzongke. See this link to download the model, and for more information.
