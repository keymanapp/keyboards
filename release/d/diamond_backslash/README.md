diamond\ (lite) keyboard
==============

Description
-----------
A lite version of the diamond\ mirroring keyboard layout. Features not available: backspace on caps lock key, secondary keyboard layout toggle.

Read as "diamond backslash"

Head to the repository homepage to learn more about this keyboard layout.
