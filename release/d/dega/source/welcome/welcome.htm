Start Using Dega Keyboard

Dega keyboard is created for indigenous people in the Central Highlands of Vietnam, to preserve their writing, this Dega keyboard can be used by Êđê, Jơrai, Kơ’Ho, Mơnông, Bahnar and many other indigenous ethnic groups.

For how to type certain characters other than the usual Latin characters, please refer to the following bullet points.

Upper Case

  • E + E = Ê
  • E + X = Ĕ
  • E + E + X = Ê̆
  • E + ` = È
  • E + E + ` = Ề
  • U + U = Ư
  • U + X = Ŭ
  • U + U + X = Ư̆
  • U + W = Ư
  • U + W + X = Ư̆
  • U + ` = Ù
  • U + U + ` = Ừ
  • I + X = Ĭ
  • Y + X = Y̆
  • O + O = Ô
  • O + X = Ŏ
  • O + O + X = Ô̆
  • O + ` = Ò
  • O + O + ` = Ồ
  • O + W = Ơ
  • O + W + X = Ơ̆
  • O + W + ` = Ờ
  • A + A = Â
  • A + X = Ă
  • A + A + X = Â̆
  • A + ` = À
  • A + A + ` = Ầ
  • C + C = Č
  • B + B = Ƀ
  • N + N = Ñ
  • D + D = Đ

Note: You can choose to release the Shift key after pressing the first letter with it to get a capital letter, just like when you press and hold it.

Lower Case

  • e + e = ê
  • e + x = ĕ
  • e + e + x = ê̆
  • e + ` = è
  • e + e + ` = ề
  • u + u = ư
  • u + x = ŭ
  • u + u + x = ư̆
  • u + w = ư
  • u + w + x = ư̆
  • u + ` = ù
  • u + u + ` = ừ
  • i + x = ĭ
  • y + x = y̆
  • o + o = ô
  • o + x = ŏ
  • o + o + x = ô̆
  • o + ` = ò
  • o + o + ` = ồ
  • o + w = ơ
  • o + w + x = ơ̆
  • o + w + ` = ờ
  • a + a = â
  • a + x = ă
  • a + a + x = â̆
  • a + ` = à
  • a + a + ` = ầ
  • c + c = č
  • b + b = ƀ
  • n + n = ñ
  • d + d = đ

Keyboard Layout

Desktop

Default (unshifted)

Shift

Phone

Default (unshifted)

This is what is shown by default.

Shift

Press the Shift key once to be able to type one capital letter.

Shifted lock

Press the Shift key twice to be able to type more capital letters. Press the Shift key again to go to the default layer (abc).

Numeric

Press the number key (123) to go to the layer with numerals, punctuations and symbols

More Symbols

Press the symbol key to reveal more symbols including currency and brackets.
