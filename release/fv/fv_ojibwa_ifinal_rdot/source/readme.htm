ᐊᓂᔑᓇᐯᒧᐏᣙ (Ojibwa, i-finals, right w-dot) Keyboard

This keyboard is designed for the ᐊᓂᔑᓇᐯᒧᐏᣙ (Ojibwa) language of the Eastern Subarctic region of Canada. This is part of a set of keyboards from FirstVoices.

This keyboard uses i-finals and the right w-dot. For i-finals and left w-dot, see the ᐊᓂᔑᓇᐯᒧᐎᣙ (Ojibwa, right w-dot) keyboard.

For a-finals, see the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa, a-finals, left w-dot) keyboard or the ᐊᓂᔑᓇᐯᒧᐏᓐ (Ojibwa, a-finals, right w-dot) keyboard.

For Western-finals, see the ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓂᓂᒧᐏᐣ (Severn Ojibwa, right w-dot) keyboard.
