X̱aayda-X̱aad Kil Keyboard

This keyboard is designed for the X̱aayda-X̱aad Kil language of the BC Coast region of Canada. This is part of a set of keyboards from FirstVoices.

  • To type the underline accent use the Semicolon key ; — g̱ is typed g then ;
  • To type the acute accent use the Backquote key ` — á is typed a then `
  • To type the grave accent use the Shift+Backquote keys — à is typed a then Shift+`
  • To type the circumflex accent use the Shift+6 keys — ĝ is typed g then Shift+6
  • You can type the Original English punctuation by holding down the Right-Alt or Option key. The Semicolon ; is typed Right-Alt+; (Windows) / ⌥ Option+; (Mac)
    • NOTE: When using Google Docs, Slash / is typed Right-Alt+Shift+/ (Windows) / ⌥ Option+Shift+/ (Mac)
  • Opening and closing quotes. For Mac users, Right-Alt is either of the Option keys.
    • Single: ‘ Shift+\ — ’ '
    • Double: “ Shift+[ — ” Shift+]
    • Single: ‹ Right-Alt+Shift+9 — › Right-Alt+Shift+0
    • Double: « Right-Alt+0 — » Right-Alt+0
