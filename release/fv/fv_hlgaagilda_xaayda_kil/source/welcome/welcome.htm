X̱aayda-X̱aad Kil Keyboard

This keyboard is designed for the X̱aayda-X̱aad Kil language of the BC Coast region of Canada. This is part of a set of keyboards from FirstVoices.

  • To type the underline accent use the Semicolon key ;. g̱ is typed g then Semicolon.
  • To type the acute accent use the Grave key `. á is typed a then Grave.
  • To type the grave accent use the Shift+Grave key. à is typed a then Shift+Grave.
  • To type the circumflex accent use the Shift+6 key. ĝ is typed g then Shift+6.
