• The lowline has been assigned to the underscore key _ (shift+hyphen key). To type the regular underscore, hold down the Right-Alt or Option key then press shift + hyphen key.
  • To type g̲, type g + _
  • For any double or triple letters that have the lowline, type all the letters followed by the underscore _.
    • To type h̲h̲, type hh + _
    • To type w̲u̲h̲, type wuh + _

Unicode Fonts

The following fonts have been installed on your computer: Aboriginal Sans, Aboriginal Serif. Please use these fonts when typing in your language as they have been specially built to represent your language as accurately as possible.
