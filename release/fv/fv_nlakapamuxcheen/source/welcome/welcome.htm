• The lowline has been assigned to the colon key ;. To type the regular colon, hold down the Right-Alt or Option key then press the colon key.
  • To type g̲, type g + ;
  • For any double or triple letters that have the lowline, type all the letters followed by the colon key.
    • To type h̲h̲, type hh + ;
    • To type w̲u̲h̲, type wuh + ;
