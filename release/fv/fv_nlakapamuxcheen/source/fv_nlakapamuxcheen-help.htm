• The lowline has been assigned to the underscore key _ Shift+-. To type the regular underscore, hold down the Right-Alt (Windows) or ⌥ Option (Mac) key then Shift+-.
  • To type g̲, type g then Shift+-
  • For any double or triple letters that have the lowline, type all the letters followed by the underscore _.
    • To type h̲h̲, type hh then Shift+-
    • To type w̲u̲h̲, type wuh then Shift+-
