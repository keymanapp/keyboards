• To type the acute accent use the Backquote ` (next to the 1 key)
    • é is typed e then Backquote `
  • The ḵ and x̱ characters have been assigned to the [ and ] keys.
    • ḵ is typed with the left bracket key [
    • x̱ is typed with the right bracket key ]
  • All changed punctuation can type their original value by holding down the Right-Alt key (Win) or Option key (Mac).
    • The left bracket [ is typed Right-Alt+[ (Win) or Option+[ (Mac).

Unicode Fonts
