• All accents are typed after the base character: á is typed a then Backquote key ` (next to the 1 key)
  • To type the acute accent use the Semicolon key ; — é is typed e then Backquote key `
  • All changed punctuation can type their original value by holding down the Right-Alt or Option key. The Backquote key ` is typed Right-Alt+Backquote (Windows) Option+Backquote (Mac).

    • Unicode Fonts

    The following fonts have been installed on your computer: Aboriginal Sans, Aboriginal Serif. Please use these fonts when typing in your language as they have been specially built to represent your language as accurately as possible.
