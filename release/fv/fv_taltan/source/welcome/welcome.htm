• All accents are typed after the base character — ā is typed a then Slash /
    • To type the macron accent ̄ use the Slash key / — ē is typed e then /
    • To type the underline accent _ use the Left-bracket key [ — s̱ is typed s then Left-bracket [
  • You can type the English value of the punctuation keys by holding down the Right-Alt or Option key. The Slash / is typed Right-Alt+/ (Windows) / ⌥ Option+/ (Mac).
    • NOTE: When using Google Docs, Slash / is typed Right-Alt+Shift+/ (Windows) / ⌥ Option+Shift+/ (Mac)
  • Opening and closing quotes. For Mac users, Right-Alt is either of the Option Keys.
    • single: ‘ ` — ’ '
    • double: “ Shift+[ — ” Shift+]
    • single: ‹ Right-Alt+Shift+9 — › Right-Alt+Shift+0
    • double: « Right-Alt+9 — » Right-Alt+0
