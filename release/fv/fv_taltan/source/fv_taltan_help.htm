• All accents are typed after the base character — ā is typed a then Slash /
  • To type the macron accent ̄ use the Slash key / — ē is typed e then Slash /
  • To type the underline accent _ use the Left-bracket [ — s̱ is typed s then Left-bracket [
  • You can type the English value of the punctuation keys by holding down the Right-Alt or Option key. The Slash / is typed Right-Alt+Slash (Windows) Option+Slash (Mac).
