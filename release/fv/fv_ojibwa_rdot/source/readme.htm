ᐁᓂᔑᓇᐯᒧᐏᓐ (Ojibwa, a-finals, right w-dot) Keyboard

This keyboard is designed for the ᐁᓂᔑᓇᐯᒧᐏᓐ (Ojibwa) language of the Eastern Subarctic region of Canada. This is part of a set of keyboards from FirstVoices.

This keyboard uses a-finals and the right w-dot. For a-finals and left w-dot, see the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa, a-finals, left w-dot) keyboard.

For i-finals, see the NAME (left w-dot) keyboard or the NAME (right w-dot) keyboard.

For Western-finals, see the ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓂᓂᒧᐏᐣ (Severn Ojibwa, right w-dot) keyboard.
