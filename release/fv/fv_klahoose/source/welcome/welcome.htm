This keyboard is designed for the Éy7á7juuthem language of the BC Coast region of Canada.

If square boxes are displayed instead of characters when using this keyboard (and in the keyboard layouts below), please read our troubleshooting guide.

Desktop Keyboard Layout

Default

  • The "Default" keymap above shows how to type the following characters in Éy7á7juuthem.
    • ƛ — [ left bracket on English layout
    • θ — ] right bracket
    • ə — ; semicolon
    • ʔ — / slash
  • To type a combining accent, first type the base alphabet character (such as a), and then type the accent key as listed below.
    • Acute accent (◌́) — Backtick ` (next to 1)
      • Example: e + ` = é
    • Combining glottal apostrophe (◌̓) — Apostrophe '
      • Example: m + ' = m̓
    • Caron (◌̌) — Backslash \
      • Example: c + \ = č
  • You can type the original English value of the keys by holding down the Right-Alt or Option key.
    • Example: The left bracket [ is typed Right-Alt+[ (Windows) or Option+[ (Mac).

Shifted

  • The "Shifted" keymap above shows characters in Éy7á7juuthem that can be typed with SHIFT key combinations.
    • ɛ — SHIFT + e
    • ʊ — SHIFT + u
    • I — SHIFT + i
    • ɔ — SHIFT + e
    • æ — SHIFT + a
    • ɬ — SHIFT + l
    • χ — SHIFT + x
    • w̥ — SHIFT + v
  • You can type the original English value of the keys by holding down the Right-Alt or Option key.
    • Example: Capital A is typed RightAlt + a (Windows) or Option + a (Mac).

Typing raised characters

  • To type a raised character, first type the base alphabet character (such as y, and then type the Caret ^ (Shift + 6).
    • Raised ᶿ: θ + ^ or t + ^
    • Raised ʷ: w + ^
    • Raised ʸ: y + ^

The following Unicode fonts have been installed on your computer: Aboriginal Sans, Aboriginal Serif. Please use these fonts when typing in your language as they have been specially built to represent your language as accurately as possible.
