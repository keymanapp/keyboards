Desktop Layout

  • Accents are typed after the base letter — ā is typed a then Left-Bracket [
    • To type the macron accent use the Left-Bracket key [ — ō is typed o then Left-Bracket [
    • To type the circumflex accent use the Right-Bracket key ] - ê is typed e then Right-Bracket ]
    • To type the caron accent use the Backslash key \ — š is typed a then Backslash
  • You can type the original English punctuation by holding down the Right-Alt or Option key. The Semicolon ; is typed Right-Alt+Semicolon (Windows) Option+Semicolon (Mac).
