• The syllabics used to write ᓀᐦᐃᔭᐍᐏᐣ (Plains Cree) are typed by entering their Long Vowel equivalents.
    • ᓀᐦᐃᔭᐍᐏᐣ would be typed as 'nehiyawewin'
  • In the case of syllables with multiple syllable finals, type the left bracket key [ to keep the syllables separate
    • ᐸᐦᐠᐍᓯᑲᐣ would be typed 'pahk[wesikan'
    • The narrow no-break space is typed shift+space
  • Quotation marks
    • double guillemets: « Semicolon — » Period
    • double: " Right-Alt+Semicolon — " Right-Alt+Period
