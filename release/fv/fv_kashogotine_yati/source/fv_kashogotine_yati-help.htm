Keyboard Layout

  • The special characters of K'áshogot'ı̨nę́ Yatı̨́ have been assigned to various keys
  • All accents are typed after the base character: á is typed a then Left-Bracket [
  • To type the acute accent use the Left-Bracket [
    í is typed ı then [
  • To type the nasal accent use the Right-Bracket ]
    ǫ is typed o then ]
  • To type the acute and nasal accents simultaneously use the backslash key \
    ǫ́ is typed o then \
    • The combined acute and nasal accents can also be typed by typing the accents individually in either order
  • The schwa ə is typed by pressing the 3 key
  • The barred l ł can be typed using the semi-colon key ;
  • The dotless i ı is typed using the i key
  • The glottal stop ɂ is typed using the slash key /
  • All changed keys can type their original value by holding down the Right-Alt or Option key. To get the original value of the left-bracket [ type Right-Alt+Left-Bracket (Windows) Option+Left-Bracket (Mac).
