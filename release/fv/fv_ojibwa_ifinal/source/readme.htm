ᐊᓂᔑᓇᐯᒧᐎᣙ (Ojibwa, i-finals, left w-dot) Keyboard

This keyboard is designed for the ᐊᓂᔑᓇᐯᒧᐎᣙ (Ojibwa) language of the Eastern Subarctic region of Canada. This is part of a set of keyboards from FirstVoices.

This keyboard uses i-finals and the left w-dot. For i-finals and right w-dot, see the ᐊᓂᔑᓇᐯᒧᐏᣙ (Ojibwa, i-finals, right w-dot) keyboard.

For a-finals, see the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓇᐯᒧᐏᓐ (Ojibwa, right w-dot) keyboard.

For Western-finals, see the ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓂᓂᒧᐏᐣ (Severn Ojibwa, right w-dot) keyboard.
