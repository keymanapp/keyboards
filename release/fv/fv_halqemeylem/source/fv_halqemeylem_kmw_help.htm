• To type the acute accent use the Slash key / — é is typed e then Slash /
  • To type the grave accent use the Backquote key ` — è is typed e then Backquote `
  • The original English punctuation keys can be typed by holding down the Ctrl+Alt or Option key. The Semicolon ; is typed Ctrl+Alt+Semicolon (Windows) Option+Semicolon (Mac).
