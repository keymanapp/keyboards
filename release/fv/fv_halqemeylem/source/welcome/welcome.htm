• All tone marks are typed after the base character.
    • á is typed a then Slash /
  • To type the high tone use the Slash key / — é is typed e then Slash /
  • To type the mid tone use the Backquote key ` — è is typed e then Backquote `
  • The original English punctuation keys can be typed by holding down the Right-Alt key (Win) or Option key (Mac). The Semicolon ; is typed Right-Alt+; (Win) or Option+; (Mac).
  • Opening and closing quotes. For Mac users, Option is instead of the Right-Alt Key.
    • single: ‘ Shift+` (Backquote) — ’ Apostrophe
    • double: “ Shift+-Hyphen — ” Shift+= Equals
    • single ‹ Right-Alt+Shift+9 — › Right-Alt+Shift+0
    • double « Right-Alt+9 — » Right-Alt+0
