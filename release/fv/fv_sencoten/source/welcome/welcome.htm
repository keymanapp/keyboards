Desktop keyboard

To type in SENĆOŦEN on Mac or PC, turn CAPS LOCK on. To type a special character, type the base character and then type an accenting key. (Ṯ is typed T then Semicolon ;)

  • To type the acute accent ´ use the Apostrophe key ' — Ć is typed C then Apostrophe '
  • To type the slash accent use the Slash key / — Ⱥ is typed A then Slash /
  • To type the dash accent use the Hyphen key - — Ŧ is typed T then Hyphen -
  • There are two options to type the low line accent:
    • Use the Semicolon key ; — Ṉ is typed N then Semicolon ;
    • Use the Underscore key _ — Ṉ is typed N then Shift+_ Underscore
  • To type cedilla (glottal stop) press Shift+, (Comma) OR turn on Caps Lock and type Comma , (Cedilla is treated as capitalized comma.)
  • Opening and closing quotes. For Mac users, Right-Alt is either of the Option Keys.
    • double: « Left-Bracket [ — » Right-Bracket ]
    • single: ‹ Shift+[ Left-Bracket — › Shift+] Right-Bracket
  • The original English punctuation keys can always be typed by holding down the Right-Alt key (Windows) or Option key (Mac).
    • The slash / is typed Right-Alt+/ (Windows) or Option+/ (Mac).

Unicode Fonts

The following fonts have been installed on your computer: Aboriginal Sans, Aboriginal Serif. Please use these fonts when typing in your language as they have been specially built to represent your language as accurately as possible.
