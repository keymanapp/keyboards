• The underlined letters s̲, z̲, t̲s̲, and d̲z̲ each have their own key. To type the underline accent separately, use the Backquote key ` (left of the 1 key).
  • To type the acute accent use the Slash key / — á is typed a then Slash /
  • You can type the original English punctuation by holding down the Right-Alt or Option key. The Semicolon ; is typed Right-Alt+Semicolon (Windows) Option+Semicolon (Mac).
  • Opening and closing quotes. For Mac users, Right-Alt is either of the Option Keys.
    • single: ‘ Shift+Backquote — ’ Apostrophe
    • double: “ Shift+Hyphen — ” Shift+Equals

    • double « Right-Alt+9 — » Right-Alt+0
