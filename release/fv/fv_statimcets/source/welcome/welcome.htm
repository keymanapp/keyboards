• All accents are typed after the base character: é is typed e then Semicolon ;
  • To type the acute accent use the Semicolon key ; — á is typed a then Semicolon ;
  • To type the apostrophe accent use the Apostrophe key ' — k̓ is typed k then Apostrophe '
  • You can type the original English punctuation marks by holding down the Right-Alt or Option key. The Slash / is typed Right-Alt+Slash (Windows) Option+Slash (Mac).
    • NOTE: When using Google Docs, / is typed Shift+Right-Alt+/
  • Opening and closing quotes. For Mac users, Right-Alt is either of the Option Keys.
    • single: ‘ Right-Alt+7 — ’ Right-Alt+8
    • double: “ Shift+Hyphen — ” Shift+Equals
    • single ‹ Right-Alt+Shift+9 — › Right-Alt+Shift+0
    • double « Right-Alt+9 — » Right-Alt+0
