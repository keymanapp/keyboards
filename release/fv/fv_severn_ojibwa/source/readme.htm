ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa, left w-dot) Keyboard

This keyboard is designed for the ᐊᓂᔑᓂᓂᒧᐎᐣ (Severn Ojibwa) language of the Eastern Subarctic region of Canada. This is part of a set of keyboards from FirstVoices.

This keyboard uses Western-finals and the left w-dot. For Western-finals and right w-dot, see the ᐊᓂᔑᓂᓂᒧᐏᐣ (Severn Ojibwa, right w-dot) keyboard.

For i-finals, see the NAME (left w-dot) keyboard or the NAME (right w-dot) keyboard.

For a-finals, see the ᐊᓂᔑᓇᐯᒧᐎᓐ (Ojibwa, left w-dot) keyboard or the ᐊᓂᔑᓇᐯᒧᐏᓐ (Ojibwa, right w-dot) keyboard.
