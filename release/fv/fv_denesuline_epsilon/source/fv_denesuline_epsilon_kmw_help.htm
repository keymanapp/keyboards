• To type the high-tone accent use the Backquote key ` — á is typed a then Backquote `
  • To type the nasal accent use the Right-Bracket key ] — ą is typed a then Right-Bracket ]
  • To type the high-nasal accent use the Equals key = — ą́ is typed a then Equals =
  • You can type the original English punctuation by holding down the Right-Alt or Option key. The Semicolon ; is typed Right-Alt+Semicolon (Windows) Option+Semicolon (Mac).
