• The image above shows the letters of Nəxʷsƛ̓ay̓əmúcən that have been assigned to individual keys. The original English value of the keys can still be typed by holding the Right-Alt key (Win) or Option key (Mac)
    • ŋ is typed with the g key
    • g is typed with the Right-Alt+g (Win) or Option+g (Mac)
  • To type the glottal accent, type the base character first, followed by the d key.
    • q̕ is typed q then d
  • To type acccented vowels, type the Semicolon key ; first, followed by the base vowel.
    • ; a types á
    • ; e types é
    • ; f types ə́
    • ; i types í
    • ; u types ú
  • The following letters are typed by typing the Semicolon key ; first, followed by another key.
    • ; l types l
    • ; q types qʷ
    • ; z or ; x types x̣ʷ
  • The glottalized barred lambda ƛ̕ is typed u i
  • Some additional symbols are listed here.
    • . . types … (ellipsis)
    • ; \ types √
    • ; = types ≠
