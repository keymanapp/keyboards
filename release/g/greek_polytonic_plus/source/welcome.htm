Start Using Greek Polytonic Plus

 

This package includes one keyboard file, Greek Polytonic Plus.  This keyboard supports both ancient Greek (code grc) and modern (code el). Use the drop-down menu to access the one you want when installing the package. If you use both, install the package twice. The package does not include any fonts.  Most computers already have fonts installed that supply almost all the characters covered by this keyboard. Times New Roman and Calibri have them all except the angle brackets, which are found in Cambria Math; these fonts are typically installed with Windows.  On Mac OS, Times New Roman is perhaps the best choice; it lacks only capital Jot, capital Kai ligature, some of the archaic letters and the angle brackets. If you need additional fonts, New Athena Unicode is complete, as the Brill Typeface (it omits the Coptic letters).  There are many other polytonic Greek fonts available which can easily be found on line. 

 

After installing, click the Keyman icon and choose Greek Polytonic Plus.  You can also use Windows Key-Spacebar to move among installed keyboards; Windows Key-Ctrl-Spacebar takes you back to the keyboard that was previously in use.  On a Mac, use Control-Spacebar to select the previous input source, or Control-Option-Spacebar for the next input source in the Input menu. The globe/fn key can be configured to switch between the two most recently used languages. To access the on-screen keyboard in Windows, type Alt-Shift-K; on Mac OS, choose On-Screen Keyboard under Keyman in the Input Source menu.

[use Option-ι on Mac]

Macron and breve are not available with breathings and accents; see documentation for full discussion..
