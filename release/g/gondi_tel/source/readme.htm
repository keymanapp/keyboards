This Gondi (Telugu) Keyman keyboard has been designed for use on physical keyboards (desktops and laptops) as well as hand-held touch devices (such as smart phones and tablets). It works on most of the currently available Operating Systems: Windows, Linux, MacOS, Android, and iOS. The keyboard had been optimized for the Gondi language spoken in the erstwhile district of Adilabad, which is in the state of Telangana, in central India. The ISO code for this language is wsg. The BCP-47 code for the script is wsg-Telu.
