# Geezword Keyboard

Keyboard for Tigrinya, Amharic and Geez, Tigre and Blin languages (ትግርኛ፡ አማርኛ፡ ግእዝ፡ ትግረ፡ ብለን)

It uses a phonetic keyboard, and it includes the GeezMahtemStandard custom font.
