Start Using Gandhari

Gandhari Sanskrit keyboard developed for Latin transliteration of Gāndhārī language written in the Kharoṣṭhī script.

This keyboard makes heavy use of "deadkeys". These keys are described in the downloadable documentation below. Where the keyboard mentions the CTRL key, the LEFT CTRL key should be used. Where the keyboard mentions the ALT key, the RIGHT ALT key should be used.

Documentation for this keyboard is available here: gandhari-keyboard_keyman.pdf
