Gondi (Devanagari) keyboard

Copyright (C) SIL International

This Gondi (Devanagari) Keyman keyboard has been designed for use on physical keyboards (desktops and laptops) as well as hand-held touch devices (such as smart phones and tablets). It works on most of the currently available Operating Systems: Windows, Linux, MacOS, Android, and iOS. The keyboard had been optimized for the Gondi languages spoken in Central India. The ISO 639-3 codes for these languages are gon, gno, esg, wsg, mrr, emu, fmu and mut.
