This Gondi (Devanagari) Keyman keyboard has been designed for use on physical keyboards (desktops and laptops) as well as hand-held touch devices (such as smart phones and tablets). It works on most of the currently available Operating Systems: Windows, Linux, MacOS, Android, and iOS. The keyboard had been optimized for the Gondi languages spoken in the districts of Yavatmal, Chandrapur, Gadchiroli and Aheri in the state of Maharashtra. The ISO codes for these languages are esg, wsg and mrr. The BCP-47 codes for these language-script combinations are esg-Deva, wsg-Deva and mrr-Deva.
