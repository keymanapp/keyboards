This Gondi (Gunjala) Keyman keyboard has been designed for use on physical keyboards (desktops and laptops) as well as hand-held touch devices (such as smart phones and tablets). It works on most of the currently available Operating Systems: Windows, Linux, MacOS, Android, and iOS. The keyboard had been optimized for the Adilabd Gondi language spoken in Central India (ISO 639-3 code: wsg).
