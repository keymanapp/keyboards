Overview

This keyboard is designed for Biblical Hebrew, a Semitic language of the ancient Near East and the language of the Hebrew Bible.

This keyboard layout works best with a QWERTY (English) keyboard. It mimics the QWERTY (English) layout, not the modern Greek layout. It uses standard Unicode fonts.

Final mem, pe, nun and tsadi

The final forms of kaf, mem, pe, nun and tsadi are generated by the keyboard layout when the Space, Tab or Enter keys are typed after the letter.

Additional characters

The ` key is a deadkey, that allows additional characters to be typed.

Technical Information

System Requirements

It is recommended that you use an English QWERTY hardware keyboard with this keyboard.

Unicode Version

This keyboard complies with Unicode 5.1

Keyboard Authorship

His effort has assisted greatly in enabling people to communicate in Koine Greek.
