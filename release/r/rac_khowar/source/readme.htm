Rachitrali-Khowar Keyman Keyboard

Rehmat Aziz Chitrali Linguist at Khowar Academy Chitral Pakistan, 2017-08-14

Introduction

This keyboard was developed by Rehmat Aziz Chitrali for keyboarding the Khowar language in the Arabic Script. The Rachitrali-Khowar keyboard is designed to work with the Scheherazade font. However, you can use this keyboard with any extended Arabic script Unicode font.
