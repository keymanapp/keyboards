Start Using Russian Mnemonic R

Russian Mnemonic R 1.0.1.

A font must be installed to support this keyboard. There are number of fonts which support the Cyrillic script.

Visit  GitHub.com/dotland/mnemonic-kb-ru  for extensive documentation on using the keyboard.

This keyboard uses eight dead keys, such as: S s C c J j \ /, where S is Shift+s or just s with Caps Lock turned on. Press one of these keys and then another key to produce the desired output. For instance, press S and then the Space to get cyrillic С; press S and then h to get cyrillic Ш etc. The complete combinations are as follows.
