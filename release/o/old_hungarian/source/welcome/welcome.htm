Start Using Old Hungarian

Keyboard for Old Hungarian. Desktop layout based on xkeyboard-config's "Old Hungarian (for ligatures)" keyboard layout by András Tisza (2013) and Viktor Kovács (2017, 2018, 2020).

Zero Width Joiner is available on the key that says Lig (Ligatúrákhoz),
the Right to left mark to available on the key that says JB (Jobbról-balra)
and the Left to right mark is available on the key that says BJ (Balról-jobbra)
