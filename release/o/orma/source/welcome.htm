How to use the Orma keyboard

Most special characters are produced by typing ; then a letter.

type ;n to get ñ

type ;c to get ç

type ;a to get ā

type ;e to get ē

type ;i to get ī

type ;o to get ō

type ;u to get ū

type ;d to get ɗ

type ;k to get ƙ

type ;p to get ƥ

type ;t to get ƫ

type ; then the upper case letter to get

ÑÇĀĒĪŌŪƊƘƤƬ


For the hook c with cedilla

type [c to get ƈ̧

type [C to get Ƈ̧


Open/close quotes

type < to get ‘

type > to get ’

type << to get “

type >> to get ”

type |< to get <

type |> to get >
