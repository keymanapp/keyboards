Start Using Old Hungarian (Carpathian Highlands)

Keyboard for Old Hungarian. This keyboard is easier for people who use the Slovak keyboard layout.

Desktop layout based on xkeyboard-config's "Old Hungarian (Carpathian Highlands)" keyboard layout. This keyboard does not provide a mobile layout. The related old_hungarian keyboard provides a suitable mobile layout.

Desktop Keyboard layout

Zero Width Joiner is available on the key that says Lig (Ligatúrákhoz),
the Right to left mark to available on the key that says JB (Jobbról-balra)
and the Left to right mark is available on the key that says BJ (Balról-jobbra)
