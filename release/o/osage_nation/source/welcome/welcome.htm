The Osage (Classic) Keyman Keyboard Package

Font

This is a keyboard for typing the Osage language in the pre-2016 orthography.

The Osage (Classic) Keyman keyboard is designed to work any Osage script font. It comes with the Noto Sans Osage font. Other Osage fonts, such as Pawhuska, are available from Osage Nation.
