The Osage Keyman Keyboard Package

Font

The Osage Keyman keyboard is designed to work any Osage script font. It comes with the Pawhuska font. This and other Osage fonts are available from Osage Nation.

Keyboard

This keyboard layout is designed for the Osage language in the Osage script. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.
