Ó tissi keyboard
==============

© 2020-2021 Indigenous Language Network

Version 2.0

Description
-----------

The Ó tissi keyboard is for use by the Ó tissi/Achumawi [acv] language. The On-Screen Keyboard can be viewed to see how to use the keyboard.
