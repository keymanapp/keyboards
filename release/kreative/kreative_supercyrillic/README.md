Kreative SuperCyrillic keyboard
==============

Version 1.1

Description
-----------
Easily type Cyrillic letters for Russian, Belarusian, Ukrainian,
Tatar, Bashkir, Kazakh, Kyrgyz, Yakut (Sakha), Tajik, Uzbek,
Azerbaijani, Mongolian, Serbian, Macedonian, and Bulgarian.
