Halang

The Halang Telex Style Keyboard.

The Halang Telex keyboard allows for the entry of 18 additional characters that cannot be inputted using the Vietnamese keyboard.

ă ĕ ĭ ŏ ŭ â̆ ê̆ ô̆ ơ̆
Ă Ĕ Ĭ Ŏ Ŭ Â̆ Ê̆ Ô̆ Ơ̆

It was designed to enable the input of Halang vowel characters based on the Vietnamese Telex input method.
