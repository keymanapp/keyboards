Start Using Halang Keyboard

The Halang keyboard allows for the entry of 18 additional characters that cannot be inputted using the Vietnamese keyboard.

ă ĕ ĭ ŏ ŭ â̆ ê̆ ô̆ ơ̆ Ă Ĕ Ĭ Ŏ Ŭ Â̆ Ê̆ Ô̆ Ơ̆


How to Input Keys.

â : Type "a" key twice.

ê : Type "e" key twice.

ô : Type "o" key twice.


ă : Type "a", plus a "w" or "q" to produce the " ă ".

ĕ : Type "e", plus a "q" to produce the "ĕ".

ĭ : Type "i", plus a "q" to produce the "ĭ".

ŏ : Type "o", plus a "q" to produce the "ŏ".

ŭ : Type "u", plus a "q" to produce the "ŭ".


â̆ : Type "a" key twice, plus a "q" to produce the " â̆ ".

ê̆ : Type "e" key twice, plus a "q" to produce the " ê̆ ".

ô̆ : Type "o" key twice, plus a "q" to produce the " ô̆ ".


à : Type "a", plus a "f" to produce the " à ".

è : Type "e", plus a "f" to produce the " è ".

ì : Type "i", plus a "f" to produce the " ì ".

ò : Type "o", plus a "f" to produce the " ò ".

ù : Type "u", plus a "f" to produce the " ù ".


ơ, ư : Type "o" or "u" followed by a "w" to produce the " ơ " or " ư ".

ươ: Type "uo" followed by a "w" to produce " ươ ".
