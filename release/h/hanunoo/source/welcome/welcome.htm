Hanunoo Keyboard

Basic Hanunoo (ᜱᜨᜳᜨᜳᜢ) Keyboard for Philippines languages, included Hanunoo, Tagalog, Buhid, etc. This keyboard covered all characters on the Unicode Standard 15.0. Other additional characters can be added in the future if necesarry. There is no standard keyboard layout yet for the Hanunoo Keyboard at the moment, so this keyboard can be one of the alternative options.

This keyboard layout designed based on suggestions from the user group in social media. If there any other suggestions about the layout design, please let me know.

+

The font used along with this keyboard is Noto Sans Hanunoo. In case you have not install it on your machine, you can simply download the font from the Google Noto website and install it.

The keyboard on this version can be installed on Windows, MacOS and Linux. It also provide the on screen keybaord.

This keyboard is part of the Kairaga.com keyboard development program. Please send your thoughts to info@kairaga.com
