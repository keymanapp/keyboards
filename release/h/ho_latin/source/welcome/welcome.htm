Ho Latin Keyboard te cheleka oloḱa:

Desktop lagit́ Rule ko (Phone reaḱ Rule ko latar paa rea)

Consonant ko

k; → ḱ
c; → ć
z; → t́
p; → ṕ
w; → ṕ

t; → ṭ
d; → ḍ
r; → ṛ

n; → ṅ
j; → ń

Diacritic menaḱ Consonant ko (ḱ, ć, t́, ṕ, ṭ, ḍ, ṛ, ṅ, ń eman) do, en Letter tayom Semicolon (;) Key otaè re hućoḱa. Example lekate: k tayom ; (k;) ota kee re ḱ hućoḱa. Enlekage, d tayom ; (d;) ota kee re ḍ hućoḱa. Maraṅ Ol ó enkage, K tayom ; (K;) otaè re Ḱ hućoḱa.

Exception ko

t́ do z tayom ; (z;) otaè re hućoḱa, t tayom ; otaè redo ṭ sekar hućoḱa. Typing re tílavroḱkâ mente ṕ do, p tayom ; otaè reó hućoḱa anḍoḱ w tayom ; otaè reó hućoḱ gea. ń do j tayom ; (j;) otaè re hućoḱa, n tayom ; otaè redo ṅ sekar hućoḱa.

Vowel ko

a; → á
a;; → â
a;;; → ã
a;;;; → ã́
a;;;;; → à
a~ → ã
a` → à
a~` → ã́
Eṭaḱ Vowel Letter (e, i, o, u) ko ó enka ge oloḱa.

Symbol ko

Semicolon (;)„ Backtick (`)„ Tilde (~), nen apia Symbol do ALT ota akaa te en Symbol ko otaè re hućoḱa. Example lekate:
ALT + ; → ;
ALT + ` → `
ALT + SHIFT + ~ → ~
Bar dau Comma (,) ota kee re „ hućoḱa, sama Comma lagit́ ALT + Comma (,) ota daeoḱa.

Letter suba re Gár

Chiminaṅ Letter ko suba re Gár ol daeoḱa ALT + SHIFT + -(Hyphen) ota kee re.
Example lekate:
n + ALT + SHIFT + - → ṉ

Phone reaḱ Rule ko

Keyboard Sanarai


Chetan paa lenga paa Number ko loḱ „ kin otaè re Diacritic menaḱ Letter ko oloḱa.
Example lekate:
"a" tayom mit́sa otaè re "á" hućoḱa.
"a" tayom bar dau ota kee re "à" hućoḱa.
"a" tayom mit́sa ota kee re "ã" hućoḱa.
"a" tayom bar dau ota kee re "â" hućoḱa.
"a" tayom mit́sa , anḍoḱ mit́sa ota kee re "ã́" hućoḱa.

"n" tayom otaè re "ń" hućoḱa.
"n" tayom otaè re "ṅ" hućoḱa.
"k", "c", "t", "p", "ṭ" tayom otaè re "ḱ", "ć", "t́", "ṕ" hućoḱa.
