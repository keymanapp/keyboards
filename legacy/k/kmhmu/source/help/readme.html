Kmhmu 2008 Keyboard (Lao orthography)

This package includes the Kmhmu 2008 keyboard and a compatible font Kmhmu OT.

Keyboard layout follows the Lao standard, with some minor differences to simplify entry of Kmhmu text.

See Using Kmhmu 2008 (in the Start - All Programs menu) or the On Screen Keyboard for layout details.

Note that standard Lao Unicode (OpenType) fonts do not permit all character sequences found in Kmhmu.
