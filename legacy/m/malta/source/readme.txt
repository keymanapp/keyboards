Maltese Keyman Keyboard
Created by Fabian van de l'Isle

Accented letters are formed by typing the appropriate letter, followed by the
key listed below for the appropriate accent mark. Note that you will need an
appropriate iso-8859-3 compatible font to view these glyphs correctly.

accent	accent			affected
key	name			letters
----------------------------------------
`	grave			A E  I  O U
^	circumflex		ACEGHIJ OSU
:	diuresis (umlaut)	A E  I  O U
-	bar (stroke)		    H
-	breve			   G      U
'	acute			A E  I  O U
.	dot			 C G I     Z
~	tilde			       N
,	cedilla			 C       S

If you need an unaccented letter followed by one of the above punctuation marks,
simply type the punctuation mark twice. This Taveultesoft keyboard is good for
typing Maltese or Esperanto. Other languages are not officially supported by
the iso-8859-3 standard. However, meaning Western European languages can be written
using this standard, including English, French, and German.

To install this keyboard completely, you also need to do the following:

1) select 'Define Languages...'
2) click on 'Add...'
3) under 'Name', type 'Maltese'
4) under 'Language ID', type '03a'
5) under 'Sublanguage ID', type '01'
6) click on 'OK'
7) click on 'Install keyboard...'
8) click on 'Local'
9) locate and select the file 'malta.kmx'
10) click on 'Open'


Fonts for this codepage can be downloaded from:
<http://www.esperanto.be/tiparoj.html>
