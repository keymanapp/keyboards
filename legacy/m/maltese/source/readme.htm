Usage

Launch Keyman and left-click its icon on the Windows taskbar to select Maltese.  Now you can start typing:

Vowels
'a 'e 'i 'o 'u 'y   →   á é í ó ú ý
"a "e "i "o "u "y   →   ä ë ï ö ü ÿ
`a `e `i `o `u     →   à è ì ò ù  
^a ^e ^i ^o ^u     →   â ê î ô û  
 
Consonants
'c 'g 'h 'z       →   ċ ġ ħ ż    

Press an accent key twice to output the accent.

System Requirements

This keyboard is used with Keyman 5 or later.

  • Windows 2000 users only need to install Keyman 5 and the keyboard.
  • Users of Windows 95/98/ME or NT 4.0 can only use this keyboard in Word 97 or later and WordPad 98 or later but they should update their system before installing Keyman 5 and the keyboard.
  • You also need WordLink for Word 97/2000/XP under Windows 95/98/ME.

Installation

  1. Download and install Keyman and whatever you need to update your system.  If you use Word 97/2000/XP under Windows 95/98/ME then you should also get WordLink.
  2. Double-click maltese.kmp to install it for Keyman.
