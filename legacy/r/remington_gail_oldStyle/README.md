Remington GAIL (SIL) keyboard
==============

© SIL International
Modified by : Danish Khan


Description
-----------

This is a modified version of Remington Gail (SIL) keyboard
where Arabic numbers are implemented in numpad keys and
matra ि  is being implemented to be typed before instead of after.
This modification is done for old school typists who are used to typing
on kruti dev or devlys keyboards.
