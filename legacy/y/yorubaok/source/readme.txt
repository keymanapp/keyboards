Notes for using the Yoruba Keyman keyboard
============================================

1. This keyboard was designed by Adebusola Onayemi and James Kass for use with the YorubaOK.ttf and similarly coded fonts.

2. For typing Yoruba, please see the accompanying Keyboard Layout image.

3. A Yoruba keyboard for Unicode-standard coded fonts is being developed by Andrew Cunningham - for details please contact him at <lang.support@gmail.com>.

All enquiries regarding keyboard layout and Yoruba language should be directed either to Adebusola Onayemi at <webmaster@learnyoruba.com>, or to Andrew Cunningham at the address given above, and not to Tavultesoft.
