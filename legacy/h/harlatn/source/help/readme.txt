Harari (Lâtîn harfi) Unicode Keyboard Layout

© State Library of Victoria and the Australian Saay Harari Association 2009

This keyboard layout was designed for use with the 
revised Harari Latin orthography adopted by the Harari 
Community Language school in Melbourne, Australia. 
The Latin script is used to teach the younger Harari 
children their own language.                  

The fonts Charis SIL Comapct and Doulos SIL Compact, released under the 
Open Font License have been included with the keyboard.
