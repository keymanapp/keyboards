Tamil keyboard for Keyman Desktop

This keyboard layout is designed for Tamil. It includes an on screen keyboard which can be viewed by clicking on the Keyman icon and selecting the On Screen Keyboard menu item.

New: This keyboard now supports the Indian Rupee symbol: ₹. Use Shift+4 to type it.

Release History
