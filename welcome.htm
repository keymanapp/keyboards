Tawrã-Kaman Keyboard

An optimized keyboard for the Tawrã (Digaro Mishmi/Taraon) and Kaman (Miju Mishmi) languages. Designed by NEILAC, Guwahati on request by these communities.

Desktop Instructions (PC)

To type special characters, type the base letter first, then the modifier key:

CharacterKeys to press
ü / Ü[ / Shift + [
ə / Ə] / Shift + ]
ñ / Ñn + \ / N + \
High Tone (á)Vowel + ` (Backquote)
Low Tone (à)Vowel + ~ (Shift + Backquote)
Mid Tone (ā)Vowel + ` twice
Circumflex (â)Vowel + ^ (Shift + 6)
Nasal (ã)Vowel + \ (Backslash)
Typing Symbols: To type a standard backslash \, press the key twice. To type brackets [ ], press the key twice.

Mobile Instructions

  • Smart Tones: Tap a vowel, then tap a tone mark from the top row.
  • Auto-Lowercase: The keyboard returns to lowercase automatically after typing one capital letter.
  • Caps Lock: Double-tap the Shift key to lock the keyboard in Uppercase mode. Tap Shift again to unlock.
