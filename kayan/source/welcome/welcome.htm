Welcome to the Kayǎn Keyboard

Key sequences

Type in the order that Kayǎn is hand written.

  1. vowel
  2. vowel change mark
  3. tone mark

Fonts

Here is a list of fonts that work well for typing Kayǎn on a computer:

  • Andika
  • Brill
  • Calibri
  • Cambria
  • Charis SIL
  • Consolas
  • Doulos SIL
  • Gentium
  • Segoe UI

Phones with Android 6 and above also work well.

Special Issue

All fonts do not display the U+0300 and U+0301 sequence correctly for Kayǎn. There is a font feature that has been proposed to help with this problem.
